## 1. "Escreve sobre redes sociais"

### **Versão A - E-commerce**
```
Trabalhas para uma loja online de produtos artesanais que quer aumentar vendas através das redes sociais. És especialista em marketing digital para pequenas empresas. Escreve um guia prático sobre como usar Instagram e Facebook para vender produtos artesanais. Formato: artigo de 800 palavras com 5 estratégias concretas, incluindo exemplos de posts e hashtags relevantes.
```

### **Versão B - Consultoria**
```
És consultora de recursos humanos que quer posicionar-se como especialista no LinkedIn. Como estratega de personal branding para profissionais liberais, cria um plano de conteúdo para LinkedIn focado em temas de gestão de equipas. Formato: calendário editorial mensal com 12 tópicos de posts, formato e call-to-action para cada um.
```

### **Versão C - Educação**
```
Trabalhas numa escola secundária que quer educar pais sobre os riscos das redes sociais para adolescentes. Como psicólogo educacional especializado em tecnologia e adolescência, desenvolve um guia informativo sobre como supervisionar o uso de redes sociais pelos filhos. Formato: folheto de 2 páginas com sinais de alerta, dicas práticas e recursos úteis para pais.
```

## 2. "Faz um texto sobre alimentação saudável"

### **Versão A - Blog Familiar**
```
Escreves para um blog de maternidade com foco em famílias portuguesas ocupadas que querem comer melhor. Como nutricionista especializada em alimentação infantil, escreve um artigo sobre como preparar lanches saudáveis para crianças em idade escolar. Formato: post de blog de 600 palavras com 8 receitas fáceis, tempo de preparação e valores nutricionais.
```

### **Versão B - Empresa Corporativa**
```
Trabalhas numa empresa de tecnologia com 200 funcionários que quer promover hábitos saudáveis no escritório. Como especialista em wellness corporativo, cria um manual de alimentação saudável no local de trabalho. Formato: e-book de 10 páginas com meal prep, snacks de escritório e dicas para reuniões com comida.
```

### **Versão C - Fitness**
```
Trabalhas num ginásio local que quer educar os seus membros sobre nutrição pós-treino. Como personal trainer certificado em nutrição desportiva, desenvolve um guia de alimentação para maximizar os resultados do treino. Formato: infográfico A4 com timing de refeições, alimentos recomendados e 3 receitas de batidos proteicos.
```

## 3. "Escreve um artigo sobre turismo em Portugal"

### **Versão A - Turismo Familiar**
```
Escreves para um website de viagens familiares que promove destinos portugueses para férias com crianças. Como blogger de viagens especializado em turismo familiar, cria um guia completo sobre as melhores praias portuguesas para famílias com crianças pequenas. Formato: artigo de 1200 palavras com 10 praias, comodidades, atividades para crianças e dicas práticas.
```

### **Versão B - Turismo Cultural**
```
Escreves para uma revista de turismo cultural dirigida a visitantes estrangeiros interessados em história portuguesa. Como historiador e guia turístico certificado, desenvolve um roteiro cultural de 3 dias por Lisboa focado no património medieval e dos Descobrimentos. Formato: itinerário detalhado com horários, preços, restaurantes tradicionais e curiosidades históricas de cada local.
```

### **Versão C - Turismo Rural**
```
Escreves para uma plataforma de turismo rural que quer promover o interior de Portugal junto de portugueses da cidade. Como especialista em ecoturismo e desenvolvimento rural, escreve sobre experiências autênticas de turismo rural na região do Douro. Formato: reportagem de 800 palavras sobre quintas, atividades sazonais, gastronomia local e alojamentos únicos.
```

## Framework CPAO - Resumo

**C - Contexto:** Quem é o cliente/empresa? Qual a situação?  
**P - Papel:** Que especialista/profissional está a criar o conteúdo?  
**A - Ação:** O que exatamente deve ser criado?  
**O - Output:** Formato, extensão e elementos específicos do resultado final

Aqui estão algumas perguntas específicas e impactantes para demonstrar o poder do NotebookLM com as tuas fontes sobre AI in Marketing:

## **Perguntas para Audio Overview Demo:**

_Para mostrar como sintetiza informações de múltiplas fontes_

**1. Pergunta Estratégica:**

```
Com base nos dados do McKinsey e nas tendências do Google, qual é o ROI real que as empresas estão a conseguir com AI em marketing? Compare isso com o caso da Nike.
```

**2. Pergunta Prática:**

```
Quais são as 5 aplicações de AI em marketing que Gary Vaynerchuk recomenda e que também aparecem como high-performing no relatório McKinsey?
```

**3. Pergunta Ética:**

```
Que dilemas éticos específicos sobre AI em marketing são mencionados nos papers académicos e como é que a Nike está a lidar com esses mesmos problemas?
```

## **Perguntas para Chat/Q&A Demo:**

_Para mostrar análise profunda e cruzamento de dados_

**4. Pergunta Comparativa:**

```
Compare as previsões do Google sobre AI marketing trends para 2025 com os dados atuais de adoção que o McKinsey apresenta. Onde estão as discrepâncias?
```

**5. Pergunta de Implementação:**

```
Se eu fosse CEO de uma empresa de e-commerce, qual seria o meu roadmap de 6 meses para implementar AI em marketing, baseado nestas fontes?
```

**6. Pergunta de Resistência:**

```
Quais são os principais obstáculos éticos e práticos à implementação de AI em marketing mencionados em todas as fontes? Como ultrapassá-los?
```

## **Perguntas "Wow Factor":**

_Para impressionar a audiência_

**7. Pergunta de Síntese:**

```
Cria um business case de 2 minutos para convencer um board de diretores a investir em AI marketing, usando dados específicos destas 5 fontes.
```

**8. Pergunta Controversa:**

```
Gary Vaynerchuk diz X sobre AI, mas os papers académicos alertam para Y. Onde está a verdade e qual o meio termo responsável?
```

**9. Pergunta de Futuro:**

```
Com base em todas estas fontes, como será o marketing em 2030? Que empresas vão sobreviver e quais vão desaparecer?
```

**10. Pergunta Técnica:**

```
Explica como a Nike usa AI para personalização sem violar princípios éticos mencionados nos papers académicos. É possível replicar esta abordagem?
```

## **Pergunta "Grand Finale":**

_Para terminar com impacto_

**11. Pergunta Integradora:**

```
Se tivesses de escrever um artigo de 1000 palavras sobre "O Estado Real da AI em Marketing em 2025" usando apenas estas fontes, qual seria o teu outline e as 3 conclusões principais?
```
