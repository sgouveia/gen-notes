# Exercícios de Prompt Engineering para Content Writers

*Formação sobre IA Generativa - Sessão de Prompt Engineering*  
*Duração: 4 horas | Audiência: Content Writers*

---

## Cenário 1: E-commerce Sustentável

### **Situação:**
Trabalhas numa loja online de produtos sustentáveis. A equipa quer implementar IA para acelerar a criação de conteúdo mantendo a autenticidade da marca.

### **Parte I – Prompts Básicos**

#### **1. Template de descrição de produto:**
**Situação concreta:** A tua loja vai lançar uma nova linha de detergentes em pastilha. Precisas de um template que funcione para os 8 produtos desta linha.

**Prompt sugerido:**
```
Cria um template de descrição para detergentes em pastilha sustentáveis.
Inclui: título, 3 benefícios principais, características técnicas básicas, e call-to-action.
Usa placeholders [NOME_PRODUTO], [FRAGÂNCIA], etc.
Tom: profissional mas acessível.
```

#### **2. Boas práticas para IA:**
**Situação concreta:** O teu chefe quer implementar IA na equipa de copywriting mas tem receios sobre autenticidade da marca.

**Prompt sugerido:**
```
És consultor de marketing digital especializado em sustentabilidade.
Lista 5 boas práticas para copywriters usarem IA sem perder autenticidade da marca.
Cada prática: 1-2 frases máximo.
```

### **Parte II – Prompts Complexos**
*Escolhe 2 das seguintes opções:*

#### **Opção A - Email carrinho abandonado:**
**Situação:** Cliente "Maria, 38 anos" abandonou carrinho com shampoo sólido de €12,90 + condicionador de €9,90. É a terceira vez que visita o site.

**Prompt sugerido:**
```
Cliente Maria (38 anos) abandonou carrinho com shampoo sólido €12,90 + condicionador €9,90.
Terceira visita ao site.
Escreve email recuperação (assunto + corpo) para enviar amanhã.
Foca nos benefícios familiares e ambientais.
Máximo 150 palavras.
```

#### **Opção B - Landing page:**
**Situação:** Vais lançar uma nova escova de dentes de bambu (€3,50) direcionada a pais preocupados com plástico e saúde oral dos filhos.

**Prompt sugerido:**
```
Produto: escova dentes bambu €3,50
Público: pais preocupados com plástico e saúde oral infantil
Visitantes vêm do Facebook.
Cria: headline, 3 benefícios, 1 objeção respondida, call-to-action.
```

#### **Opção C - Posts Instagram educativos:**
**Situação:** Tens de publicar 5 posts no Instagram esta semana sobre "como reduzir plástico na cozinha", promovendo subtilmente os teus produtos.

**Prompt sugerido:**
```
5 posts Instagram sobre "reduzir plástico na cozinha"
Cada post: dica prática + sugestão subtil de produto sustentável
Tom: educativo, não promotional
Inclui emoji e hashtag sugeridas.
```

## Cenário 2: Blog de Tecnologia

### **Situação:**
Trabalhas para um blog de tecnologia que publica 3 artigos por semana. O teu editor quer que uses IA para acelerar a produção, mas mantendo qualidade e originalidade.

### **Parte I – Prompts Básicos**

#### **1. Outline de artigo:**
**Situação concreta:** Próxima semana sai o iPhone 16. Cria prompt para outline de artigo "Primeiras impressões" (1200 palavras, SEO-friendly).

**Prompt sugerido:**
```
Artigo "iPhone 16 - Primeiras Impressões" para blog de tecnologia.
1200 palavras, otimizado para SEO.
Inclui: introdução, specs principais, comparação com iPhone 15, prós/contras, conclusão.
Público: entusiastas de tecnologia.
Palavra-chave principal: "iPhone 16 review"
```

#### **2. Boas práticas jornalismo tech:**
**Situação concreta:** Lista 5 regras para usar IA em jornalismo tech sem comprometer credibilidade.

**Prompt sugerido:**
```
És editor experiente de publicação tecnológica.
Lista 5 regras essenciais para jornalistas tech usarem IA mantendo credibilidade.
Foca em verificação de factos, originalidade e ética jornalística.
Cada regra: máximo 2 linhas.
```

### **Parte II – Prompts Complexos**
*Escolhe 2 das seguintes opções:*

#### **Opção A - Newsletter semanal:**
**Situação:** Cria prompt para newsletter com resumo das 5 notícias tech mais importantes da semana.

**Prompt sugerido:**
```
Newsletter semanal para subscritores interessados em tecnologia.
Resumo das 5 notícias tech mais importantes desta semana.
Para cada notícia: título, resumo (50 palavras), por que é importante.
Tom: informativo mas acessível.
Inclui call-to-action para ler artigos completos no blog.
```

#### **Opção B - Artigo comparativo:**
**Situação:** Prompt para comparar 3 smartphones em lançamento (specs, preços, público-alvo).

**Prompt sugerido:**
```
Artigo comparativo: iPhone 16 vs Samsung Galaxy S24 vs Google Pixel 8
Compara: especificações, preços, fotografia, bateria, público-alvo ideal.
800 palavras, estrutura clara com secções.
Conclusão: recomendação para cada tipo de utilizador.
Tom: objetivo e técnico mas acessível.
```

#### **Opção C - Preparação de entrevista:**
**Situação:** Prompt para gerar 10 perguntas para entrevista com CEO de startup de IA.

**Prompt sugerido:**
```
Entrevista com CEO de startup de IA (empresa faz chatbots para e-commerce).
10 perguntas que interessem aos leitores de blog tech.
Mistura: perguntas técnicas, visão de futuro, desafios do mercado, conselhos para empreendedores.
Evita perguntas genéricas sobre IA.
```

## Cenário 3: Agência de Comunicação

### **Situação:**
A tua agência ganhou 3 clientes novos: restaurante, clínica dentária e loja de roupa vintage. Precisas de criar conteúdo rapidamente para todos.

### **Parte I – Prompts Básicos**

#### **1. Brief adaptável:**
**Situação concreta:** Cria prompt que transforme informações básicas do cliente num brief de conteúdo completo.

**Prompt sugerido:**
```
Transforma estas informações básicas num brief de conteúdo completo:
- Tipo de negócio: [SECTOR]
- Público-alvo: [DEMOGRAFIA]
- Objetivos: [METAS]
- Orçamento mensal: [VALOR]

Gera: tom de voz, temas de conteúdo, calendário editorial mensal, métricas a acompanhar.
```

#### **2. Definição de tom de voz:**
**Situação concreta:** Prompt para definir personalidade de marca e tom de comunicação para qualquer tipo de negócio.

**Prompt sugerido:**
```
Define a personalidade e tom de voz para [TIPO_NEGÓCIO].
Público-alvo: [DEMOGRAFIA]
Valores da marca: [VALORES]

Resultado: 3 adjetivos que descrevem a marca, tom de comunicação, exemplos de linguagem a usar/evitar.
Formato: guia prático para copywriters.
```

### **Parte II – Prompts Complexos**
*Escolhe 2 clientes:*

#### **Cliente A - Restaurante:**
**Situação:** 5 posts Facebook para promover menu semanal (cada post diferente).

**Prompt sugerido:**
```
Restaurante familiar português, especialidade pratos tradicionais.
5 posts Facebook para promover menu desta semana.
Cada post: abordagem diferente (ingredientes, tradição, preço, experiência, chef).
Inclui call-to-action para reservas.
Tom: acolhedor e familiar.
```

#### **Cliente B - Clínica dentária:**
**Situação:** Artigo blog "Cuidados dentários no verão" (800 palavras, linguagem acessível).

**Prompt sugerido:**
```
Artigo para blog de clínica dentária: "Cuidados dentários no verão"
800 palavras, linguagem acessível a famílias.
Aborda: hidratação, cuidados na praia, alimentação, urgências dentárias em viagem.
Inclui 2-3 calls-to-action subtis para marcar consulta.
Tom: profissional mas amigável.
```

#### **Cliente C - Loja vintage:**
**Situação:** Descrições de 10 peças únicas para Instagram Stories.

**Prompt sugerido:**
```
Loja de roupa vintage, público jovem 20-35 anos.
10 descrições criativas para peças únicas (vestidos, casacos, acessórios anos 70-90).
Para Instagram Stories: máximo 25 palavras por peça.
Destaca: época, estilo, como combinar.
Tom: descontraído, fashion-forward.
```
