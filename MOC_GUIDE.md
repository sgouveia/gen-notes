# MOC Guide

A **MOC (Map of Content)** is a hub note that links related notes together. Unlike folders, a note can appear in multiple MOCs simultaneously. MOCs are how you navigate this vault.

## Creating a new MOC

1. Name it `MOC - <Topic>.md`
2. Write a short description of what it covers
3. Link to the notes it contains, grouped by category
4. Link to parent/sibling MOCs under a "Ligacoes" section
5. Add the new MOC link in its parent MOC

Only create a MOC when you have 5+ related notes. Keep the hierarchy to 2-3 levels max.

## Template

```markdown
# MOC - <Topic>

Brief description.

## <Category>
- [[Note 1]]
- [[Note 2]]

## Ligacoes
- [[MOC - Exercise Bank]]
```

## Adding a new note

When you create a note, link it from the relevant MOC. This is the only maintenance MOCs need — add the link when you create the note.
