# Fundamentos de Desenvolvimento Web: HTML, CSS e Tailwind

O curso oferece uma introdução abrangente ao desenvolvimento web, centrado nos fundamentos de HTML, CSS e Tailwind CSS. Ao longo das sessões, os formandos irão adquirir as bases necessárias para construir e estilizar páginas web, aprendendo a estruturar conteúdo, aplicar estilos e utilizar layouts responsivos (para desktop e mobile). Com uma forte componente prática (os formandos aprendem fazendo) alicerçada em fundamentos teóricos essenciais, o curso permitirá aos formandos serem capazes de criar projectos web do zero, culminando num projecto final que poderá ser publicado online.


## No final do curso, os formandos serão capazes de:

- Compreender o funcionamento básico da internet, incluindo a comunicação entre servidores e clientes;
- Escrever código HTML semântico para estruturar páginas web de forma adequado às convenções atuais;
- Aplicar estilos CSS para melhorar o design visual de uma página, utilizando tanto CSS tradicional como a framework Tailwind CSS;
- Criar layouts responsivos, adaptados a diferentes dispositivos e tamanhos de ecrã;
- Validar o código HTML/CSS e publicar projectos online, utilizando plataformas de alojamento como o GitHub Pages ou o Netlify;
- Desenvolver um website completo e funcional, implementando boas práticas de acessibilidade e optimização para motores de busca (SEO).

## Módulo 1 - Introdução à Internet e à Web (2 horas)
### Tópicos:
- Como funciona a internet (servidores, clientes, DNS)
- O papel de HTML/CSS no desenvolvimento web
- Ferramentas: Google Chrome, VS Code, Dev Tools

### Atividade:
- Inspecionar um site real através das DevTools do browser

## Módulo 2 - Fundamentos de HTML (3 horas)

### Tópicos:
- Sintaxe de HTML: tags, atributos, nesting
- Estrutura: cabeçalhos, parágrafos, listas, imagens, links
- Gestão de ficheiros: pastas, caminhos

### Atividade:
- Criar uma página "about me" simples 

## Módulo 3 - Fundamentos de CSS (3 horas)
### Tópicos:
- Sintaxe de CSS: seletores, propriedades, valores
- Estilizar texto, cores e espaçamento
- O box model (margin, padding, border)
- **Introdução ao Tailwind CSS (30 mins)**
	- Utility-first vs CSS tradicional
	- Usar o Tailwind via CDN
	- Exemplo: `class="bg-blue-500 p-4"` vs. escrever CSS manualmente

### Atividade:
- Estilizar a página "about me" com CSS puro **e** recriar uma secção com Tailwind

## Módulo 4 - Deep Dive de HTML (2 horas)
### Tópicos:
- Tags semânticas: header, nav, article
- Formulários: input, button, label
- divs vs spans

### Atividade:
- Adicionar um formulário de contacto à página "about me"

## Módulo 5 - Layouts com CSS (2 horas)

### Tópicos:
- Fundamentos de Flexbox (display: flex, alinhamento)
- **Tailwind para layouts**:
	- Usar classes `flex`, `justify~` e `items-`
	- Prefixos responsivos (`md:flex`, `lg:text-xl`)
- Media queries

### Atividade:
- Construir uma barra de navegação responsiva com Flexbox (CSS puro **ou** Tailwind)

## Módulo 6 - Lab de Tailwind (2 horas)

**Projeto**: criar um componente "Cartão de Perfil" responsivo:

- Usar classes do Tailwind para:
	- Grids (`grid`, `grid-cols-3`)
	- Espaçamento (`p-4`, `m-2`)
	- Hover effects (`hover:bg-gray-100`)
- Comparar com a implementação em CSS puro

## Módulo 7 - Projeto Final (4 horas)

**Scope**: Construir um site de 1-2 páginas (ex: portfólio, landing page de uma empresa)
**Requisitos**:
- HTML semântico
- Design responsivo para dispositivos móveis
- **Escolha de Estilização**:
	- CSS puro **ou** Tailwind
	- Misturar ambos (ex: CSS para layout, Tailwind para botões)
**Exemplos de projetos**:
- Menu de café com preços
- Portfólio pessoal com projetos e formulário de contacto

## Módulo 8 - Publicação e Próximos Passos (2 horas)

### Tópicos:
- Validar HTML/CSS com ferramentas do W3C
- Publicar via GitHub Pages/Netlify
- Noções básicas de SEO (meta tags, alt text)
- **O mundo real**
	- Tailwind e outros frameworks utilitários
	- JavaScript, React, bibliotecas de componentes

### Atividade:
- Publicar o projeto final online
