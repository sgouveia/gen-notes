# Checklist - Validação de Conteúdo B-Corp

## Validação de Conteúdo Gerado por AI

### Checklist Completo

Use esta checklist antes de publicar qualquer conteúdo gerado por AI.

---

## 1. Verificação Factual ✅

### Precisão de Dados
- [ ] Todos os números e estatísticas são correctos?
- [ ] Datas e cronologias estão precisas?
- [ ] Nomes de parceiros/organizações estão correctos?
- [ ] Localizações são precisas?

### Claims e Afirmações
- [ ] Todas as claims são suportadas por evidência?
- [ ] Não há exageros ou hiperboles injustificados?
- [ ] Limitações são reconhecidas honestamente?
- [ ] Projeções futuras são realistas?

### Contexto B-Corp
- [ ] Requisitos de certificação são precisos?
- [ ] Processo B-Corp está correctamente descrito?
- [ ] Comunidade B-Corp é representada com precisão?

---

## 2. Alinhamento com Valores ✅

### Missão da Empresa
- [ ] Conteúdo reflecte nossa missão específica?
- [ ] Valores fundamentais são demonstrados?
- [ ] Diferenciadores são destacados?

### Pilares B-Corp
- [ ] Governança é mencionada quando relevante?
- [ ] Impacto ambiental é quantificado?
- [ ] Impacto social é demonstrado?
- [ ] Transparência é evidenciada?

### Consistência de Marca
- [ ] Tom de voz é consistente?
- [ ] Mensagens são alinhadas com comunicação anterior?
- [ ] Estilo visual segue guidelines?

---

## 3. Clareza para Públicos Externos ✅

### Acessibilidade
- [ ] Linguagem é compreensível para audiência-alvo?
- [ ] Acrónimos são explicados na primeira utilização?
- [ ] Conceitos complexos são simplificados?
- [ ] Jargão técnico é minimizado ou explicado?

### Estrutura
- [ ] Ideias fluem logicamente?
- [ ] Títulos e subtítulos são claros?
- [ ] Formatação facilita leitura?
- [ ] Pontos-chave são destacados?

### Contextualização
- [ ] Leitor não-B-Corp entende conceitos?
- [ ] Informação de fundo é fornecida quando necessário?
- [ ] Links para recursos adicionais são incluídos?

---

## 4. Tom e Formato ✅

### Adequação
- [ ] Tom é apropriado ao canal? (email vs social vs formal)
- [ ] Formato funciona para o medium?
- [ ] Duração/extensão é adequada?

### Engajamento
- [ ] Conteúdo é interessante e relevante?
- [ ] Storytelling é utilizado eficazmente?
- [ ] Visual atrai atenção?

### Profissionalismo
- [ ] Não há erros gramaticais?
- [ ] Ortografia está correcta?
- [ ] Formatação é consistente?

---

## 5. Inclusão de Exemplos de Impacto Real ✅

### Concretude
- [ ] Exemplos específicos são fornecidos?
- [ ] Dados de impacto são mensuráveis?
- [ ] Histórias são autênticas?

### Diversidade
- [ ] Diferentes tipos de impacto são mostrados? (ambiental/social/económico)
- [ ] Múltiplas partes interessadas são representadas?
- [ ] Várias geografias/contextos são incluídos quando relevante?

### Atualidade
- [ ] Exemplos são recentes?
- [ ] Progresso desde certificação é demonstrado?
- [ ] Metas futuras são mencionadas?

---

## 6. Aspectos Legais e Éticos ✅

### Direitos e Permissões
- [ ] Imagens têm licença/permisão?
- [ ] Dados de terceiros são anonimizados quando necessário?
- [ ] Citações são atribuídas correctamente?

### Transparência
- [ ] É claro quando se usa AI? (se política o exigir)
- [ ] Limitações são reconhecidas?
- [ ] Fontes são citadas?

### Responsabilidade
- [ ] Conteúdo não pode ser mal interpretado?
- [ ] Riscos são apresentados equilibradamente?
- [ ] Decisões importantes requerem consulta humana?

---

## Processo de Validação

### Passo 1: Auto-Revisão
- [ ] Criador aplica checklist
- [ ] Correcções iniciais são feitas

### Passo 2: Revisão por Pares
- [ ] Colega revisa conteúdo
- [ ] Feedback é incorporado

### Passo 3: Aprovação Final
- [ ] Responsável aprova
- [ ] Versão final é guardada

### Passo 4: Publicação
- [ ] Conteúdo é publicado
- [ ] Monitorização de feedback

---

## Nota Importante

> Esta checklist é uma ferramenta de apoio, não substitui o julgamento humano e a responsabilidade editorial.

**Em caso de dúvida:**
- Consultar colegas
- Verificar com assessoria jurídica se necessário
- Adiar publicação até ter certeza
