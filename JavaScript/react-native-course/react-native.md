# React Native

## pré-requisitos

Este curso foi criado para todos os que tenham um entendimento geral (ainda que não avançado) de React e JavaScript, e desejam potenciar essas competências para a criação de aplicações mobile.

Como pré-requisitos para retirar valor do curso, os alunos deverão:

- Estar confortáveis com os conceitos básicos de JavaScript. Ser capaz de usar *if-statements*, funções e *callbacks*, a API de *arrays*, métodos, e desenvolver pequenos programas em JavaScript.
- Estar confortáveis com os conceitos básicos de React. Já ter criado uma aplicação em React, que envolva o desenvolvimento de componentes, *rendered views*, ler e atualizar estado de um componente e a utilização de React Router ou outra ferramenta de navegação.
- Ter tido contacto com a construção de páginas *web* responsivas, que se ajustam a diferentes tamanhos e formas de ecrã, utilizando Flexbox.

Quem teve alguma experiência com as tecnologias acima, está pronto para iniciar este curso. No caso de as mesmas serem desconhecidas (ou estiverem enferrujadas), sugerimos que antes de iniciar tire algum tempo para rever ou compreender estes conceitos. Sugerimos os seguintes recursos para o fazer:

- JavaScript - https://javascript.info/first-steps
- Flexbox - https://flexboxfroggy.com/
- React - https://react.dev/learn/tutorial-tic-tac-toe

## plano curricular

No final do curso os alunos deverão ter adquirido uma visão sólida sobre o desenvolvimento *mobile* e os seus desafios específicos. Irão ainda aprofundar os seus conhecimentos de React e JavaScript. Por fim, serão capazes de desenhar uma estratégia clara e prática para a construção das suas próprias aplicações em React Native.

### Introdução a React Native (1 hora)

- Porquê aprender React Native
- Apresentação do projeto final
- Ferramentas e ambiente de desenvolvimento necessários para o curso

### Ecrãs mobile e estilização (4 horas)

- Criar e organizar ecrãs *mobile* em React Native
- Ecrãs mobile vs páginas *web*
- Adicionar conteúdo e estilo a ecrãs *mobile*
- Utilizar Flexbox para garantir que o conteúdo se adapta a qualquer dispositivo *mobile*
- Utilização de APIs nativas para melhor estilização
- Conhecer as particularidades de estilo e requisitos de *UI* exclusivas de dispositivos *mobile*.

### Routing em React Native (4 horas)

- Instalação e configuração de uma biblioteca de *routing* para React Native
- Desenvolvimento da lógica de navegação entre ecrãs
- Criação e integração de elementos de UI que permitam ao utilizador a navegação entre ecrãs.
- Erros e problemas comuns específicos de *routing* em desenvolvimento mobile.

### Gestão de estado com Redux (4 horas)

- *Setup* do Redux *toolkit* numa aplicação React Native
- Ler e escrever dados para a Redux Store global
- Entender a filosofia de *one-way data flow* de Redux e por que é segura e útil.
- Criar *reducers* de Redux que actualizam a Store.

### Funcionalidades Mobile Nativas (4 horas)

- Adicionar *push notifications* a uma aplicação React Native
- Adicionar *AsyncStorage* nativa a uma aplicação React Native
- Usar *AsyncStorage* para persistência de dados e como ferramenta de desenvolvimento
- Requisitos para a publicação de uma *app* mobile nas *app stores* da Apple e Google.

### Desenvolvimento de uma app de CRM (8 horas)

Criação de uma aplicação *mobile* para comerciais em viagem que contactam clientes diretamente e que possam criar ou atualizar os seus perfis via *smartphone*, sem necessidade de atualizar o sistema central através dos seus computadores ao final do dia. O propósito é garantir que a recolha de dados e a sua inserção no sistema acontecem simultaneamente, sem a empresa correr riscos de dados desatualizados ou ausência dos mesmos.

O desenvolvimento deste projeto será realizado em sessões presenciais com o apoio de um formador em sala de aula.
