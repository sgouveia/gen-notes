# Template: Comunicação com Pais

## Convite para Eventos (Versão Completa)

### Prompt Completo

```
Eu sou um [Introduza a sua função]. 
Com base nas informações fornecidas, redija um e-mail envolvente e informativo 
convidando os pais a participar no [nome do evento]. 

Incentive a sua participação destacando:
- Benefícios para os alunos
- Sentido de comunidade
- Detalhes claros do evento

Assunto: [Nome do evento] convite: Junte-se a nós!

Nível de escolaridade: [Inserir nível de escolaridade]

Data e hora do evento: [Introduza a data e hora]

Descrição do evento: 
[Fornecer uma breve visão geral do evento, o seu propósito e quaisquer destaques]

Público-alvo: 
[Especifique se o evento é para todos os encarregados de educação, séries específicas ou grupos específicos de alunos]

Resultado pretendido: 
[Indique o seu objetivo para a participação dos pais: maior envolvimento, construção de comunidade, apoio à aprendizagem dos alunos, etc.]

Tom: 
[Note o tom ou mensagem específica a transmitir: entusiasmado, informativo, persuasivo, etc.]

Informações adicionais opcionais: 
[Incluir quaisquer atividades especiais planeadas, se serão servidos refrescos, 
se é necessário confirmar presença ou registar-se]
```

## Ideias Bónus para Comunicação Eficaz

### 1. Relatórios de Progresso Personalizados

**Quando usar:** Fim de período, reuniões de pais, momentos de transição

**Prompt:**
```
Com base nas discussões em sala de aula e minhas observações sobre [nome do aluno], 
elabore um relatório de progresso personalizado destacando:
- Pontos fortes
- Áreas de melhoria  
- Recomendações específicas para os pais
- Metas para o próximo período
```

---

### 2. Boletins Informativos Visuais

**Quando usar:** Comunicação regular, início de mês, eventos especiais

**Prompt:**
```
Crie um boletim informativo visualmente apelativo para pais de alunos do [nível] incluindo:
- Resumo dos principais conceitos abordados este mês
- Próximos eventos importantes
- Curiosidades ou destaques
- Recursos adicionais para apoio em casa
- Formato sugerido: [email/printável/digital]
```

---

### 3. Suporte de Tradução

**Quando usar:** Famílias multilingues, comunidades diversas

**Prompt:**
```
Traduza o seguinte texto para [língua] mantendo:
- Tom acolhedor e profissional
- Terminologia educativa apropriada
- Clareza de informação importante
- Formatação original

Texto: [colar mensagem original]
```

## Dicas de Comunicação Eficaz

### Tom Adequado

**Formal (eventos institucionais):**
- Reuniões de pais
- Cerimónias de entrega de prémios
- Conferências

**Semi-formal (comunicação regular):**
- Newsletters mensais
- Atualizações de progresso
- Anúncios de atividades

**Acolhedor (comunidade escolar):**
- Eventos sociais
- Voluntariado
- Celebrações

### Elementos Essenciais

Todo email deve incluir:
- [ ] Assunto claro e atraente
- [ ] Saudação personalizada quando possível
- [ ] Propósito do contacto
- [ ] Informações práticas (data, hora, local)
- [ ] Benefícios para participação
- [ ] Call to action clara
- [ ] Contacto para dúvidas
- [ ] Despedida cordial

## Exemplos por Contexto

### Início de Ano Letivo

```
Crie um email de boas-vindas para pais de alunos do [nível] no início do ano letivo, incluindo:
- Apresentação da equipa docente
- Visão geral do programa
- Expectativas do ano
- Formas de comunicação
- Eventos importantes do calendário
```

### Preocupações Académicas

```
Redija um email sensível e construtivo para pais de [nome do aluno] sobre:
- Preocupações académicas observadas
- Estratégias de apoio implementadas
- Sugestões para apoio em casa
- Convite para reunião
```

### Celebrações de Sucesso

```
Escreva um email celebrando conquistas de [nome do aluno], destacando:
- Progresso específico
- Esforço e dedicação
- Impacto positivo na turma
- Reconhecimento dos pais
```

---

## Checklist de Qualidade

Antes de enviar comunicação gerada por AI:

- [ ] Verificar exactidão de datas e horários
- [ ] Confirmar tom adequado ao contexto
- [ ] Personalizar quando possível
- [ ] Incluir toda informação prática necessária
- [ ] Rever clareza e fluidez
- [ ] Testar links ou anexos
- [ ] Verificar destinatários
