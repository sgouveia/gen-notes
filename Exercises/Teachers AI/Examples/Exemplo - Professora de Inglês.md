# Exemplo: Professora de Inglês

## Contexto

**Professora:** Inglês do 10º ano  
**Objetivo:** Criar apresentações de slides sobre recursos literários  
**Abordagem:** Usar AI para esboço, depois adaptar manualmente

**Filosofia:** "O melhor dos dois mundos — AI ajuda a começar, eu crio exatamente o que quero"

---

## Cenário: Conflito como Recurso Literário

### Iteração 1: Prompt Básico

**Prompt inicial:**
```
Sou professora de Inglês do 10º ano e estou a elaborar uma aula sobre recursos literários. 
Forneça o texto em cada diapositivo para uma apresentação sobre o conflito na literatura.
```

**Resultado:** Básico, estrutura genérica

---

### Iteração 2: Adicionar Notas do Orador

**Prompt refinado:**
```
Sou professora de Inglês do 10º ano e estou a elaborar uma aula para os alunos sobre recursos literários. 
Crie um esboço com notas do orador para uma apresentação de diapositivos sobre o conflito na literatura.
```

**Melhoria:** Inclui notas do orador para guiar explicação

---

### Iteração 3: Especificações Detalhadas

**Prompt completo:**
```
Sou professora de Inglês do 10º ano. 
Crie o texto para um slide deck sobre o conflito como recurso literário para a aula, 
incluindo notas do orador. 

Devem existir cinco diapositivos de texto formatados com:
- Um cabeçalho para cada diapositivo
- Um conjunto de marcadores para cada diapositivo

Estrutura obrigatória:
- O primeiro diapositivo deve incluir texto sobre os objetivos da lição
- O último diapositivo deve incluir texto para uma avaliação formativa e uma atividade do aluno
```

**Especificações adicionadas:**
- Número exacto de slides (5)
- Formatação específica (cabeçalhos + marcadores)
- Conteúdo do 1º slide (objectivos)
- Conteúdo do último slide (avaliação + atividade)

---

## Adaptação Manual Final

**O que a professora fez depois:**

1. **Reorganizou** estrutura dos slides
2. **Adicionou** tipos de letra úteis
3. **Incluiu** exemplos dos textos lidos pelo turma esse ano
4. **Personalizou** com referências culturais relevantes
5. **Ajustou** linguagem ao nível específico dos alunos

**Resultado:** Apresentação única, adaptada, pronta para usar

---

## Lições Aprendidas

### Vantagens desta Abordagem

1. **Poupa tempo** — Não começar de zero
2. **Estrutura sólida** — Esqueleto bem organizado
3. **Consistência** — Formato profissional
4. **Flexibilidade** — Total liberdade para adaptar

### Cuidados Necessários

1. **Conhecimento do conteúdo** — Professor deve dominar matéria
2. **Contexto da turma** — Adaptar a experiências de leitura dos alunos
3. **Nível adequado** — Ajustar vocabulário e complexidade
4. **Verificação** — Fact-check de informação gerada

## Template Adaptável

```
Sou professor/a de [disciplina] do [nível]. 
Crie um esboço com notas do orador para uma apresentação de [N] slides sobre [tema].

Requisitos:
- Formato: [cabeçalhos/marcadores/parágrafos]
- Primeiro slide: [objectivos/introdução]
- Último slide: [avaliação/conclusão/atividade]
- Incluir: [elementos específicos]
```

---

## Checklist de Revisão

Após gerar esboço com AI:

- [ ] Verificar exactidão do conteúdo
- [ ] Adaptar exemplos à turma específica
- [ ] Ajustar nível de linguagem
- [ ] Adicionar elementos visuais
- [ ] Incluir referências culturais relevantes
- [ ] Preparar notas pessoais do orador
- [ ] Testar timing da apresentação
