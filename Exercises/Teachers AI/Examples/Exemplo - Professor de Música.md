# Exemplo: Professor de Música

## Contexto

**Professor:** Ensina bateria numa escola de música  
**Desafio:** Escrever emails a pais, colegas, alunos e newsletter mensal sobre atividades  
**Solução AI:** Templates personalizados para diferentes audiências

---

## Exemplo: Newsletter de Início de Ano

### O Prompt

```
És um professor de bateria de uma escola de música. 
Cria um template para uma newsletter de início do ano lectivo, 
com calendário de eventos para familiares, staff e restantes estudantes
```

### Por Que Funciona

**Persona definida:** "Professor de bateria"  
**Contexto claro:** "Escola de música", "início do ano lectivo"  
**Output específico:** "Template para newsletter"  
**Audiência múltipla:** Familiares, staff, estudantes  
**Conteúdo estruturado:** Calendário de eventos

### Aplicações

Este template pode ser adaptado para:
- Início de ano lectivo
- Fim de período
- Eventos especiais (concertos, audições)
- Anúncios de novos cursos
- Atualizações da escola

---

## Variáveis para Personalizar

**Informações a substituir:**
- [Nome da escola]
- [Ano lectivo]
- [Datas específicas]
- [Eventos do calendário]
- [Contactos]
- [Links relevantes]

---

## Dicas de Refinamento

**Adicionar tom específico:**
```
Torna a newsletter mais [entusiasta/formal/acolhedora]
```

**Ajustar formato:**
```
Formata como [email/HTML/documento para imprimir]
```

**Incluir elementos visuais:**
```
Sugere [imagens/gráficos/emojis] para incluir
```
