## Diferenciação por Níveis

### Nível 1: Apoio
Para alunos que necessitam de suporte adicional:

```
Crie questões de compreensão com:
- Linguagem simplificada
- Apoios visuais
- Múltipla escolha com poucas opções
- Contextos familiares
```

### Nível 2: Padrão
Para alunos com desempenho adequado ao nível:

```
Coloque questões analíticas que exijam:
- Compreensão mais profunda
- Aplicação de conceitos
- Análise de textos ou dados
- Respostas construídas
```

### Nível 3: Avançado
Para alunos com desempenho acima do nível:

```
Ofereça perguntas abertas de pensamento crítico:
- Síntese de informação
- Avaliação e julgamento
- Criatividade e originalidade
- Conexões interdisciplinares
```

---

## Variações no Formato de Resposta

Permitir que alunos escolham como demonstrar conhecimento:

- **Composições escritas** — Ensaios, relatórios
- **Apresentações** — Oral ou multimodal
- **Projetos criativos** — Arte, vídeo, design
- **Explicações orais** — Entrevistas, podcasts
- **Portefólios** — Coleção de trabalhos

---

## Variedade de Tarefas

Combinar diferentes tipos de avaliação:

- **Escolha múltipla** — Conhecimento factual
- **Respostas curtas** — Compreensão e aplicação
- **Respostas abertas** — Análise e avaliação
- **Baseadas em projetos** — Síntese e criação
- **Pares ou grupos** — Colaboração
- **Performance** — Demonstração prática

---

## Ajustes de Acessibilidade

Tornar avaliação inclusiva:

- **Tecnologia de apoio** — Leitores de ecrã, software de ditado
- **Formatos alternativos** — Áudio, vídeo, grande texto
- **Tempo extra** — Para alunos que necessitem
- **Ambiente calmo** — Espaço sem distrações
- **Instruções claras** — Passo a passo, exemplos

## Ideias Bónus

### Relatórios de Progresso Personalizados

```
Com base nas discussões em sala de aula e observações, 
utilize a ferramenta de IA para elaborar relatórios de progresso personalizados, 
destacando pontos fortes, áreas de melhoria e recomendações específicas para os pais.
```

### Boletins Informativos

```
Crie boletins informativos visualmente apelativos com:
- Resumos dos principais conceitos
- Próximos eventos
- Curiosidades
- Recursos adicionais
```

### Suporte de Tradução

```
Obtenha suporte de tradução da ferramenta de IA para facilitar 
comunicação com famílias na língua da sua preferência.
```

## Checklist de Qualidade

Antes de aplicar avaliação gerada por AI:

- [ ] Verificar exactidão de conteúdo
- [ ] Confirmar alinhamento com objetivos de aprendizagem
- [ ] Testar tempo necessário
- [ ] Revisar clareza de instruções
- [ ] Validar rubrica/chave de resposta
- [ ] Adaptar às necessidades específicas da turma
- [ ] Garantir inclusão e acessibilidade
