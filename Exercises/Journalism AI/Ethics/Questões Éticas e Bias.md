### 2. Compreender o Enviesamento (Bias)

#### Tipos de Bias

**1. Bias nos Dados**
- Dados de treino desproporcionados
- Sub-representação de grupos
- Histórico de discriminação

**2. Bias Algorítmico**
- Fórmulas que amplificam desigualdades
- Feedback loops negativos
- Falta de diversidade nas equipas de desenvolvimento

**3. Bias de Interpretação**
- Leitura enviesada de outputs
- Confirmação de preconceitos existentes
- Contexto cultural ausente

#### Identificar e Mitigar

**Checklist de Deteção:**
- [ ] Outputs representam diversidade?
- [ ] Linguagem é neutra e inclusiva?
- [ ] Fontes são variadas e credíveis?
- [ ] Análise cruzada confirma dados?

**Estratégias de Mitigação:**
1. Diversificar prompts
2. Usar múltiplas ferramentas
3. Verificação humana obrigatória
4. Feedback de comunidades afetadas

### 4. Framework Ético para Jornalistas

#### Perguntas a Fazer

**Antes de usar AI:**
1. É necessário? Há alternativas?
2. Qual o risco de erro?
3. Como verificaremos o output?
4. Revelaremos o uso de AI?

**Durante o processo:**
1. Os outputs são imparciais?
2. Respeitamos privacidade?
3. Cumprimos regulamentos?
4. Documentamos o processo?

**Após a publicação:**
1. Corrigimos erros rapidamente?
2. Aprendemos com feedback?
3. Atualizamos diretrizes?
4. Partilhamos boas práticas?
