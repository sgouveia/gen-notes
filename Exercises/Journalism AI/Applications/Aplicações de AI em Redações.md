# Aplicações de AI em Redações

## Real-World AI Applications in Newsrooms

**Duração:** 2 horas

### 1. Escrita de Notícias Automatizada

#### Exemplos de Uso
- **Relatórios financeiros:** Earnings, resultados trimestrais
- **Desportos:** Relatos de jogos, estatísticas
- **Meteorologia:** Previsões locais
- **Trânsito:** Acidentes, condições de estrada

#### Vantagens
- Velocidade na publicação
- Cobertura de múltiplas fontes simultaneamente
- Consistência no formato
- Libertar jornalistas para análise profunda

#### Prompt Exemplo
```
Escreve um artigo de 300 palavras sobre os resultados financeiros 
da [empresa] no Q3, incluindo: receitas, lucros, comparação YoY, 
comentário do CEO, e perspetivas futuras.
```

---

### 2. Recomendações e Personalização de Conteúdo

#### Ferramentas de Curadoria
- **Análise de comportamento:** O que o leitor consome
- **Personalização:** Artigos recomendados
- **Notificações push:** Conteúdo relevante
- **Newsletters personalizadas:** Seleção automática

#### Aplicação Prática
```
Com base no histórico de leitura deste utilizador sobre [tópicos], 
sugere 5 artigos relevantes do nosso arquivo.
```

---

### 3. IA nas Redes Sociais e SEO

#### Gestão de Redes Sociais
- **Geração de posts:** Adaptar artigos para diferentes plataformas
- **Agendamento:** Otimização de horários
- **Hashtags:** Sugestão automática
- **Engajamento:** Responder a comentários

#### SEO (Search Engine Optimization)
- **Headlines otimizadas:** Palavras-chave
- **Meta descriptions:** Resumos para Google
- **Estrutura de URL:** Amigável para SEO
- **Linking interno:** Sugestões de artigos relacionados

#### Prompt Exemplo
```
Cria 3 versões de headline para este artigo sobre [tema], 
otimizadas para SEO com a keyword [palavra-chave].
```

---

### 4. Imagens e Gráficos Gerados por IA

#### Aplicações Jornalísticas
- **Ilustrações:** Para conceitos abstratos
- **Gráficos:** Visualização de dados
- **Infografias:** Transformar dados complexos
- **Avatares:** Para proteção de identidades

#### Ferramentas
- Midjourney, DALL-E para imagens
- Canva AI, Adobe Firefly para design
- Flourish, Datawrapper para dados

#### Prompt Exemplo
```
Cria um infográfico sobre [tema] incluindo: 
- Dados estatísticos principais
- Timeline de eventos
- Comparativos visuais
- Estilo minimalista e informativo
```

---

## Exercícios Práticos

### Exercício 1: Escrita Automatizada
**Tarefa:** Usar AI para escrever 3 versões de uma notícia sobre resultados desportivos.
**Análise:** Comparar velocidade vs qualidade jornalística.

### Exercício 2: Personalização
**Tarefa:** Criar 3 newsletters diferentes para 3 perfis de leitores distintos a partir do mesmo conteúdo base.

### Exercício 3: SEO
**Tarefa:** Otimizar um artigo existente para SEO usando AI para headlines, meta description e keywords.
