# Ferramentas e Técnicas de Integração

## Integrating AI in Journalism: Tools and Techniques

**Duração:** 2 horas

### 1. Deep Dive in Current AI Technologies

#### Ferramentas Essenciais para Jornalistas

**Análise de Dados**
- **Flourish:** Visualização de dados
- **Google Pinpoint:** Análise de documentos
- **OCR + AI:** Extrair texto de imagens
- **ChatGPT Code Interpreter:** Analisar datasets

**Verificação de Factos**
- **ClaimReview:** Verificação estruturada
- **Similarity analysis:** Detetar plágio
- **Reverse image search:** Verificar origem de imagens
- **Cross-referencing:** Comparar múltiplas fontes

**Produção de Conteúdo**
- **Descript:** Edição de áudio/vídeo com transcrição
- **Otter.ai:** Transcrição automática
- **Grammarly:** Edição de texto
- **Hemingway Editor:** Clareza na escrita

### 3. Verificação de Informação

#### Cross-Referencing e Fact-Checking

**Processo de Verificação:**
1. **Extração:** Identificar claims no texto
2. **Pesquisa:** Buscar fontes primárias
3. **Comparação:** Cruzar informações
4. **Validação:** Confirmar com especialistas

**Prompt de Verificação:**
```
Verifica a seguinte afirmação: "[texto]"
Procura fontes primárias e indica se é factualmente correta, 
parcialmente correta, ou incorreta. Cita as fontes.
```

#### Deteção de Fake News

**Sinais de Alerta:**
- Fontes não verificáveis
- Linguagem sensacionalista
- Imagens manipuladas
- Datas inconsistentes

**Ferramentas:**
- Google Fact Check Tools
- InVID & WeVerify
- FotoForensics
- TinEye
