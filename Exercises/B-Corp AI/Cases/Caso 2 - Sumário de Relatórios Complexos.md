# Caso 2: Sumário de Relatórios Complexos

## Contexto

A equipa ESG precisa de enviar um resumo do relatório de impacto para as equipas da empresa que não trabalham diretamente na área ESG.

---

## Parte I: Criação de Prompt Básico

### Objetivo
Resumir relatório anual de impacto de forma acessível.

### Prompt

```
Sou o responsável de sustentabilidade de uma B Corp. 
Resume de forma clara e apelativa os principais insights do nosso relatório anual de impacto 2024, 
para ser partilhado com equipas que não trabalham diretamente na área ESG.
```

### Elementos do Prompt

**Persona:** Responsável de sustentabilidade  
**Documento:** Relatório anual de impacto 2024  
**Output:** Resumo claro e apelativo  
**Audiência:** Equipas não-ESG  
**Desafio:** Traduzir linguagem técnica para acessível

---

## Parte II: Desenvolvimento de Prompts Mais Complexos

### Tarefa 1: Versões por Departamento

Criar adaptações específicas para:

#### RH (Recursos Humanos)
**Foco:** Cultura organizacional, bem-estar, diversidade

**Prompt:**
```
Adapta o sumário para a equipa de RH, destacando: iniciativas de bem-estar, 
programas de diversidade e inclusão, cultura organizacional e desenvolvimento profissional.
```

#### Operações
**Foco:** Eficiência, cadeia de valor, logística sustentável

**Prompt:**
```
Adapta o sumário para a equipa de operações, focando em: eficiência operacional, 
sustentabilidade na cadeia de valor, logística verde e redução de resíduos.
```

#### Marketing
**Foco:** Branding de impacto, histórias, comunicação externa

**Prompt:**
```
Adapta o sumário para a equipa de marketing, enfatizando: histórias de impacto, 
diferenciação da marca B-Corp, testemunhos e oportunidades de comunicação.
```

### Tarefa 2: Carrossel LinkedIn

**Formato:** 5-7 slides verticais  
**Objetivo:** Engajamento em redes sociais

**Prompt:**
```
Converte o sumário num carrossel de slides para LinkedIn, com:
- Slide 1: Hook impactante
- Slides 2-6: Um insight por slide com dados
- Slide 7: Call-to-action
Tom: Profissional mas acessível
```

### Tarefa 3: Quiz Interno

**Objetivo:** Engajar funcionários e testar compreensão

**Prompt:**
```
Gera 10 perguntas para um quiz interno sobre o relatório de impacto, 
abrangendo: metas alcançadas, iniciativas principais, e próximos passos. 
Inclui respostas corretas.
```

---

## Exercício Prático

### Desafio
1. Usar prompt básico para criar sumário geral
2. Criar 3 versões departamentais
3. Desenvolver carrossel LinkedIn
4. Criar quiz de 5 perguntas

### Análise Comparativa

| Versão | Foco Principal | Tom | Formato |
|--------|---------------|-----|---------|
| Geral | Visão global | Neutro | Texto |
| RH | Pessoas | Motivacional | Email |
| Operações | Processos | Prático | PDF |
| Marketing | Brand | Inspirador | Carrossel |

### Discussão
- Como manter consistência entre versões?
- Que métricas são relevantes para cada departamento?
- Como medir eficácia da comunicação?
