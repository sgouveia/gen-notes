# Caso 4: Refinar Prompts para Ética e Clareza

## Contexto

Desenvolver FAQ sobre sustentabilidade para website da empresa, com foco em ética, clareza e acessibilidade.

---

## Parte I: Prompt Inicial (Fraco)

### O Problema

**Prompt:**
```
Cria uma FAQ sobre sustentabilidade para o nosso site.
```

### Porque é Fraco?

❌ Sem contexto de empresa  
❌ Sem definição de audiência  
❌ Sem número de perguntas  
❌ Sem especificação de tom  
❌ Sem requisitos de conteúdo  

**Resultado provável:**
- FAQ genérica e pouco útil
- Possível informação incorreta
- Tom inadequado
- Não reflecte valores B-Corp

---

## Parte II: Prompt Refinado (Forte)

### A Melhoria

**Prompt:**
```
Somos uma empresa certificada B Corp que atua no setor tecnológico. 
Cria uma FAQ com 5 perguntas e respostas sobre o nosso impacto ambiental e social. 

Requisitos:
- Usa linguagem acessível e evita jargão técnico
- Respostas devem ser factuais, breves e convidar à ação
- Foco em impacto mensurável e transparente
```

### Análise do Refinamento

| Elemento | Versão Fraca | Versão Forte |
|----------|--------------|--------------|
| **Quem** | ❌ Ausente | ✅ B-Corp tech |
| **O quê** | ❌ Vago | ✅ FAQ com 5 Q&As |
| **Tema** | ❌ Genérico | ✅ Impacto ambiental/social |
| **Tom** | ❌ Não definido | ✅ Acessível, factual |
| **Formato** | ❌ Implícito | ✅ Breve, action-oriented |
| **Valores** | ❌ Ausente | ✅ Transparência B-Corp |

---

## Exercício Prático

### Parte A: Análise Crítica

**Tarefa:** Comparar os outputs de ambos os prompts.

**Questões:**
1. Que diferenças notas no conteúdo?
2. Como o tom muda entre versões?
3. Qual é mais útil para visitantes do site?
4. Qual melhor representa valores B-Corp?

### Parte B: Iteração Adicional

Refinar ainda mais o prompt:

**Sugestões de melhoria:**
```
Adicione:
- "Inclui exemplos concretos do nosso impacto"
- "Menciona certificação B-Corp e o que significa"
- "Adiciona links para relatórios de impacto"
- "Termina cada resposta com convite ao engajamento"
```

**Prompt super-refinado:**
```
Somos uma empresa certificada B Corp no setor tecnológico desde 2020. 
Cria uma FAQ com 5 perguntas e respostas sobre o nosso impacto ambiental e social 
para o nosso website.

Requisitos:
- Linguagem acessível, sem jargão técnico
- Respostas factuais, breves (máx 3 frases cada)
- Incluir exemplo concreto em cada resposta
- Mencionar nossa certificação B-Corp e o que significa
- Terminar cada resposta com convite à ação
- Link para relatório de impacto completo

Perguntas a incluir:
1. O que é uma B-Corp?
2. Qual o nosso impacto ambiental?
3. Como apoiamos a comunidade?
4. Que metas temos para 2025?
5. Como posso contribuir?
```

---

## Checklist de Prompt Ético B-Corp

Antes de usar qualquer prompt para comunicação externa:

### Verdade e Transparência
- [ ] Informação é factual e verificável?
- [ ] Claims são suportados por dados?
- [ ] Limitações são reconhecidas?

### Clareza e Acessibilidade
- [ ] Linguagem é compreensível para audiência?
- [ ] Evita jargão desnecessário?
- [ ] Estrutura é lógica e clara?

### Valores B-Corp
- [ ] Reflete compromisso social?
- [ ] Demonstra responsabilidade ambiental?
- [ ] Mostra governança transparente?

### Impacto e Ação
- [ ] Informação convida ao engajamento?
- [ ] Oferece formas de participação?
- [ ] Inspira mudança positiva?
