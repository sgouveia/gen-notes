# Caso 1: Comunicação com Stakeholders

## Contexto

Uma B-Corp da área alimentar quer melhorar a comunicação com stakeholders no início de um novo ciclo estratégico.

---

## Parte I: Criação de Prompt Básico

### Objetivo
Criar uma newsletter de início de ano com impacto social e ambiental.

### Prompt

```
És o responsável de comunicação de uma empresa certificada B Corp no setor alimentar. 
Cria um template para uma newsletter de início de ano com:
- Destaques do último ano (impacto social e ambiental)
- Calendário de iniciativas com a comunidade e equipas
- Mensagem inspiradora da direção
```

### Elementos do Prompt

**Persona:** Responsável de comunicação B-Corp  
**Setor:** Alimentar  
**Contexto:** Início de ano / novo ciclo estratégico  
**Output:** Template de newsletter  
**Conteúdo:**
- Impacto social e ambiental
- Iniciativas comunitárias
- Mensagem de liderança

---

## Parte II: Desenvolvimento de Prompts Mais Complexos

### Tarefa 1: Versão para Investidores

**Adaptação necessária:**
- Focar em ROI social
- Métricas de impacto
- Crescimento sustentável
- Governança

**Prompt sugerido:**
```
Adapta a newsletter anterior para uma audiência de investidores, 
enfatizando: retorno do investimento de impacto, métricas ESG, 
crescimento sustentável e governança responsável.
```

### Tarefa 2: Post de Redes Sociais

**Formato:** LinkedIn/Instagram  
**Objetivo:** Highlights visuais

**Prompt sugerido:**
```
Cria um post para redes sociais com os highlights visuais da newsletter, 
usando tom inspirador e hashtags relevantes #BCorp #Sustentabilidade #Impacto
```

### Tarefa 3: Guião para Vídeo

**Formato:** Vídeo de 60 segundos  
**Objetivo:** Resumir pontos principais

**Prompt sugerido:**
```
Cria um guião para vídeo de 60 segundos que resuma os principais pontos 
da newsletter, com tom inspirador e call-to-action para engajamento.
```

---

## Exercício Prático

### Desafio
Usar o prompt básico para gerar template, depois criar as três versões alternativas.

### Análise
Comparar:
- Tom de voz em cada versão
- Tipo de informação priorizada
- Formato adequado à audiência

### Discussão
- Qual versão seria mais eficaz para cada stakeholder?
- Como manter consistência de marca entre versões?
- Que elementos são universais vs específicos?
