# Caso 5: Adaptar e Melhorar Output da IA

## Contexto

Um colaborador criou um rascunho com IA para uma apresentação sobre o movimento B-Corp. O objetivo é refinar o output para melhor adequação ao evento.

---

## O Prompt Final

```
Sou responsável de comunicação de uma B Corp portuguesa. 
Cria o texto para um slide deck de 6 diapositivos que explique os princípios das empresas B Corp, com:

- Títulos e bullet points
- Notas do orador
- Primeiro slide: objetivos da apresentação
- Último slide: chamada à ação para empresas que queiram juntar-se ao movimento
```

### Elementos do Prompt

**Persona:** Responsável comunicação B-Corp portuguesa  
**Output:** Slide deck de 6 diapositivos  
**Tema:** Princípios B-Corp  
**Formato:** Títulos + bullets + notas do orador  
**Estrutura:**
- Slide 1: Objetivos
- Slides 2-5: Conteúdo
- Slide 6: Call-to-action

---

## Checklist de Validação

### Após gerar output com AI, verificar:

#### 1. Verificação Factual ✅
- [ ] Todos os dados sobre B-Corp estão correctos?
- [ ] Certificação e requisitos são precisos?
- [ ] Exemplos são reais e verificáveis?
- [ ] Não há claims exagerados?

#### 2. Alinhamento com Valores da Empresa ✅
- [ ] Reflete nossa missão específica?
- [ ] Tom é consistente com nossa voz?
- [ ] Inclui nossos diferenciadores?
- [ ] Respeita nossa história?

#### 3. Clareza para Públicos Externos ✅
- [ ] Conceitos B-Corp são explicados?
- [ ] Linguagem é acessível a não-iniciados?
- [ ] Evita acrónimos sem explicação?
- [ ] Fluxo lógico é claro?

#### 4. Tom e Formato Adequado ✅
- [ ] Tom é profissional mas inspirador?
- [ ] Formato funciona para apresentação?
- [ ] Notas do orador são úteis?
- [ ] Duração é adequada (6 slides)?

#### 5. Inclusão de Exemplos de Impacto Real ✅
- [ ] Inclui casos de estudo?
- [ ] Dados de impacto são mensuráveis?
- [ ] Histórias são autênticas?
- [ ] Diferentes tipos de impacto (social/ambiental)?

---

## Processo de Adaptação

### Passo 1: Revisão Inicial
**Tarefa:** Ler output completo e identificar:
- Pontos fortes
- Informações incorretas
- Omissões importantes
- Oportunidades de melhoria

### Passo 2: Fact-Checking
**Tarefa:** Verificar:
- Dados B-Corp em bcorporation.net
- Estatísticas de impacto
- Requisitos de certificação
- Exemplos mencionados

### Passo 3: Personalização
**Tarefa:** Adaptar:
- Adicionar exemplo da própria empresa
- Incluir dados específicos de Portugal
- Mencionar parceiros locais
- Adaptar call-to-action ao contexto

### Passo 4: Refinamento
**Tarefa:** Melhorar:
- Simplificar linguagem complexa
- Fortalecer mensagens-chave
- Adicionar storytelling
- Otimizar para apresentação

### Passo 5: Validação Final
**Tarefa:** Usar checklist acima antes de apresentar.

---

## Template de Apresentação B-Corp (6 Slides)

### Slide 1: Objetivos
- O que é o movimento B-Corp
- Porque a certificação importa
- Como as empresas podem participar

### Slides 2-3: O Que é B-Corp
- Definição e critérios
- Processo de certificação
- Comunidade global

### Slides 4-5: Impacto e Benefícios
- Exemplos de empresas B-Corp
- Dados de impacto mensurável
- Vantagens para negócio

### Slide 6: Call-to-Action
- Próximos passos para certificação
- Recursos disponíveis
- Convite ao movimento

### Desafio
1. Usar o prompt para gerar apresentação
2. Aplicar checklist de validação
3. Identificar 3 pontos que precisam de adaptação
4. Fazer adaptações manuais
5. Preparar para apresentação de 10 minutos

### Análise

- O que se manteve do output original?
- Como o processo de validação melhorou o resultado?
