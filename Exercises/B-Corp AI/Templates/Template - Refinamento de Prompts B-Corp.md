# Template - Refinamento de Prompts B-Corp

## Estrutura para Prompts Éticos

### Framework B-Corp para Prompts

#### 1. Propósito Claro
**Definir:**
- Qual problema estamos a resolver?
- Quem é a audiência?
- Que acção queremos inspirar?

**Template:**
```
Somos uma B-Corp no setor de [setor] focada em [missão]. 
Precisamos de [tipo de conteúdo] para [audiência] que [objectivo].
```

#### 2. Contexto Específico
**Incluir:**
- História da empresa
- Valores específicos
- Certificação B-Corp (quando relevante)
- Contexto geográfico/cultural

**Template:**
```
Contexto: Empresa certificada B-Corp desde [ano], operando em [localização], 
com foco em [áreas de impacto].
```

#### 3. Requisitos de Conteúdo
**Especificar:**
- Formato exacto
- Comprimento/extensão
- Elementos obrigatórios
- Exemplos/concretos

**Template:**
```
Requisitos:
- Formato: [email/documento/apresentação]
- Extensão: [X palavras/slides/minutos]
- Incluir: [lista de elementos obrigatórios]
- Tom: [profissional/acolhedor/inspirador]
```

#### 4. Valores e Ética
**Garantir:**
- Transparência
- Clareza factual
- Acessibilidade
- Inclusão

**Template:**
```
Diretrizes:
- Usar linguagem acessível, evitar jargão
- Basear em factos verificáveis
- Incluir dados de impacto mensuráveis
- Inspirar acção positiva
```

## Exemplo Completo

### Antes (Prompt Fraco)
```
Cria conteúdo sobre sustentabilidade.
```

### Depois (Prompt Forte B-Corp)
```
Somos uma B-Corp no setor alimentar focada em agricultura regenerativa. 
Precisamos de uma newsletter mensal para clientes e stakeholders que 
demonstre nosso impacto ambiental e inspire confiança.

Contexto: Certificados B-Corp desde 2019, operações em Portugal, 
foco em redução de pegada carbónica e apoio a agricultores locais.

Requisitos:
- Formato: Newsletter email (500-700 palavras)
- Incluir: Dados do mês, história de impacto, dica sustentável, 
  próximos passos
- Tom: Profissional mas acolhedor, inspirador mas factual

Diretrizes:
- Usar linguagem acessível
- Incluir dados específicos e mensuráveis
- Mencionar parceiros locais
- Terminar com convite ao engajamento
```

## Checklist de Qualidade de Prompt B-Corp

### Antes de Enviar Prompt

#### Clareza
- [ ] Propósito é específico e mensurável?
- [ ] Audiência está claramente definida?
- [ ] Formato de output é explícito?

#### Contexto
- [ ] Empresa/certificação é mencionada?
- [ ] Setor e especialização são claros?
- [ ] Valores relevantes são incluídos?

#### Ética
- [ ] Prompt pede factualidade?
- [ ] Acessibilidade é requerida?
- [ ] Transparência é valorizada?

#### Acção
- [ ] Output deve inspirar comportamento?
- [ ] Call-to-action é implícita ou explícita?
- [ ] Engajamento é incentivado?
