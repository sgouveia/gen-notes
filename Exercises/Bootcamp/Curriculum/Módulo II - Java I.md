# Módulo II - Java I (Semanas 2-5)

## Visão Geral

**Duração:** 4 semanas  
**Foco:** Design e desenvolvimento em Java  
**Objetivo:** Aprender técnicas de design usadas por experts da indústria

## Conteúdo Programático

### 1. Java Language e Platform
- Sintaxe Java
- Tipos de dados
- Controlo de fluxo
- Tratamento de exceções
- JVM e bytecode

### 2. Programação Orientada a Objetos
- Classes e objetos
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Interfaces

### 3. Graphics Libraries
- JavaFX / Swing (introdução)
- Criação de interfaces gráficas
- Event handling
- Renderização

### 4. Java Collections
- Listas (ArrayList, LinkedList)
- Sets (HashSet, TreeSet)
- Maps (HashMap, TreeMap)
- Iterators
- Streams (introdução)

### 5. Java I/O
- File handling
- Input/Output streams
- Serialização
- Manipulação de texto

## Competências Desenvolvidas

Ao final deste módulo, os alunos serão capazes de:
- [ ] Programar em Java com fluência
- [ ] Aplicar princípios de OOP
- [ ] Criar aplicações com interface gráfica
- [ ] Manipular coleções de dados
- [ ] Ler e escrever ficheiros
- [ ] Transformar ideias em software

## Projetos Práticos

- Aplicações desktop simples
- Jogos 2D básicos
- Sistemas de gestão de dados
- Ferramentas de utilidade

## Porque Java?

- Linguagem empresarial dominante
- Forte tipagem e OOP
- Ecossistema vasto
- Base para frameworks enterprise
- Demanda no mercado de trabalho
