# Módulo I - Fundamentos (Semana 1)

## Visão Geral

**Duração:** 1 semana  
**Foco:** Fundamentos de Ciência da Computação  
**Objetivo:** Compreender como um programa é executado pela máquina

## Conteúdo Programático

### 1. Arquitetura de Computadores
- Modelo de von Neumann
- Componentes de hardware
- CPU, memória, storage
- De código a instruções binárias

### 2. Sistemas Operativos
- Funções principais do SO
- Gestão de processos
- Sistemas de ficheiros
- Linux/Unix fundamentals

### 3. Linguagens de Programação
- Sintaxe e semântica
- Compiladores vs interpretadores
- Paradigmas de programação
- Evolução das linguagens

### 4. Algoritmos e Estruturas de Dados
- Lógica de programação
- Algoritmos de ordenação
- Estruturas: arrays, listas, filas, pilhas
- Complexidade temporal

## Competências Desenvolvidas

Ao final deste módulo, os alunos serão capazes de:
- [ ] Explicar como um computador executa programas
- [ ] Compreender a relação entre código-fonte e código máquina
- [ ] Usar linha de comandos Linux básico
- [ ] Resolver problemas usando algoritmos
- [ ] Organizar dados em estruturas adequadas

## Projetos Práticos

- Configuração de ambiente de desenvolvimento
- Exercícios de lógica e algoritmos
- Primeiros programas

## Porque é Importante?

> "Arquitetura de computadores desde 1950 até hoje — o modelo é ainda o mesmo."

Estes fundamentos permitem:
- Entender erros em qualquer linguagem
- Tomar decisões técnicas informadas
- Adaptar-se a novas tecnologias
- Compreender limitações de sistemas
