# Módulo VI - JavaScript (Semanas 13-14)

## Visão Geral

**Duração:** 2 semanas  
**Foco:** Linguagem essencial do desenvolvimento web moderno  
**Objetivo:** Dominar JavaScript para criar aplicações web interativas

## Conteúdo Programático

### 1. Fundamentals of JavaScript
- Variáveis (let, const, var)
- Tipos de dados
- Funções e arrow functions
- Arrays e objetos
- ES6+ features

### 2. Web Development e DOM Manipulation
- DOM (Document Object Model)
- Selecionar elementos
- Modificar conteúdo
- Event listeners
- Criar e remover elementos

### 3. Asynchronous Programming
- Callbacks
- Promises
- Async/await
- Fetch API
- Error handling

### 4. Single Page Application (SPA)
- Conceito de SPA
- Routing
- State management (básico)
- Componentes
- Build tools (Webpack, Vite)

### 5. ES Next
- Features recentes do JavaScript
- Modules (import/export)
- Destructuring
- Spread operator
- Template literals
- Classes

## Competências Desenvolvidas

Ao final deste módulo, os alunos serão capazes de:
- [ ] Programar em JavaScript moderno
- [ ] Manipular o DOM
- [ ] Fazer chamadas assíncronas a APIs
- [ ] Criar SPAs simples
- [ ] Usar features ES6+
- [ ] Integrar frontend com backend

## Projetos Práticos

- Aplicação SPA consumindo API
- Jogo interativo no browser
- Dashboard com dados dinâmicos
- Clone simplificado de app conhecida

## A Importância do JavaScript

> "JavaScript é a única linguagem de programação do browser, ferramenta de UI por excelência de qualquer aplicação aos dias de hoje."

### Porque é JavaScript essencial?
- **Obrigatório para web:** Única linguagem nativa do browser
- **Versátil:** Frontend, backend (Node.js), mobile, desktop
- **Ecossistema gigantesco:** React, Vue, Angular, Express, etc.
- **Demanda universal:** Nenhum developer pode ignorar JS

### O Desafio do Ecossistema
- Mudanças profundas em períodos curtos
- Novas frameworks a cada semestre
- **Solução:** Conhecer a fundo a linguagem base
