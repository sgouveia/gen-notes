# Módulo V - Frameworks Java (Semanas 10-12)

## Visão Geral

**Duração:** 3 semanas  
**Foco:** Frameworks populares e desenvolvimento web enterprise  
**Objetivo:** Construir aplicações web profissionais com tecnologias industry-standard

## Conteúdo Programático

### 1. Enterprise Application Architecture
- Arquitetura em camadas
- Inversão de controlo (IoC)
- Injeção de dependências
- Configuração e properties

### 2. Java Frameworks

#### Spring
- Spring Core
- Spring Boot
- Spring Data
- Configuração automática
- Starter dependencies

#### Hibernate
- ORM (Object-Relational Mapping)
- Mapeamento de entidades
- Relacionamentos
- HQL (Hibernate Query Language)
- Lazy vs Eager loading

### 3. Frontend Programming
- HTML5 semântico
- CSS3 e responsividade
- Flexbox e Grid
- JavaScript básico
- DOM manipulation

### 4. Java Web Programming
- Tomcat server
- Spring MVC
- Thymeleaf (templating)
- Form handling
- Validation

### 5. Web Services e REST
- RESTful APIs
- JSON
- HTTP methods (GET, POST, PUT, DELETE)
- Status codes
- API documentation

## Competências Desenvolvidas

Ao final deste módulo, os alunos serão capazes de:
- [ ] Usar Spring Boot para criar aplicações
- [ ] Implementar persistência com Hibernate
- [ ] Criar APIs RESTful
- [ ] Desenvolver interfaces web
- [ ] Estruturar aplicações enterprise
- [ ] Consumir e criar web services

## Projetos Práticos

- Aplicação web completa (backend + frontend)
- API REST com documentação
- Sistema CRUD com base de dados
- Aplicação com autenticação

## Sobre Frameworks

> "A maioria dos nossos empregadores elogia sobretudo esta capacidade nos nossos alunos — de fazer as perguntas certas quando tomam contacto com novas tecnologias."

Frameworks são voláteis, mas os padrões permanecem.
