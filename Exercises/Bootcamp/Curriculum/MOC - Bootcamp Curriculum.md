# MOC: Bootcamp - Plano Curricular

## Sobre Este Plano

Plano curricular completo do bootcamp de programação, estruturado em 6 módulos ao longo de 14 semanas.

## Estrutura do Programa

### Visão Geral

**Duração Total:** 14 semanas (tempo integral)  
**Horário:** 9h às 18h (presencial)  
**Metodologia:** Educação baseada em problemas, não treino de receitas

### Filosofia Pedagógica

- **Ambiente Natural:** Questões e tarefas do mundo real
- **Crítico:** Pensamento crítico, não apenas "passar no teste"
- **Seguro:** Espaço colaborativo com feedback constante
- **Iterativo:** Tentar → Falhar → Receber feedback → Tentar novamente

## Módulos

### [[Módulo I - Fundamentos]] (Semana 1)
**Foco:** Fundamentos de Ciência da Computação  
**Tópicos:**
- Arquitetura de computadores
- Sistemas operativos
- Linguagens de programação
- Algoritmos e estruturas de dados

### [[Módulo II - Java I]] (Semanas 2-5)
**Foco:** Design e desenvolvimento em Java  
**Tópicos:**
- Java language e platform
- Programação orientada a objetos
- Graphics libraries
- Java Collections
- Java I/O

### [[Módulo III - Web e Redes]] (Semanas 6-7)
**Foco:** Internet, Web e concorrência  
**Tópicos:**
- Network Programming
- World Wide Web
- Concurrency
- Functional programming

### [[Módulo IV - Engenharia de Software]] (Semanas 8-9)
**Foco:** Padrões e práticas industriais  
**Tópicos:**
- Software engineering
- Architectural patterns
- Java environment tools
- Testing
- Relational Databases

### [[Módulo V - Frameworks Java]] (Semanas 10-12)
**Foco:** Frameworks populares e desenvolvimento web  
**Tópicos:**
- Enterprise application architecture
- Spring & Hibernate
- Frontend (HTML, CSS)
- Java web programming (Tomcat, Spring MVC, Thymeleaf)
- Web Services e REST

### [[Módulo VI - JavaScript]] (Semanas 13-14)
**Foco:** Linguagem essencial do web  
**Tópicos:**
- Fundamentals of JavaScript
- Web Development e DOM manipulation
- Asynchronous programming
- Single Page Application
- ES Next

## Fundamentos que Duram

### Porque Focamos em Fundamentos?

> "A área da programação é cumulativa. As inovações surgem em cima de outras e vão sendo construídas em novas camadas de abstração."

### Pilares Essenciais

1. **Arquitetura de Computadores (von Neumann)**
   - Desde 1950 até hoje
   - Mesmo modelo em todos os dispositivos
   - Essencial para troubleshooting

2. **Sistemas Operativos UNIX**
   - Ubíquos: smartphones, servidores, sistemas embebidos
   - Base para toda a interação computacional

3. **Algoritmia e Paradigmas**
   - Resolução de problemas agnóstica de linguagem
   - Foco em pensamento computacional
   - O que as entrevistas de emprego testam

4. **Internet e Web**
   - Protocolos e estrutura
   - Maioria do software corre na web
   - Essencial para decisões técnicas

5. **Concorrência**
   - Problema central do software moderno
   - Milhões de utilizadores simultâneos
   - Gestão de acesso a dados

6. **Bases de Dados Relacionais (SQL)**
   - Dados são o valor mais importante
   - Poucas alterações desde os anos 60
   - Fundação conservadora da indústria

7. **Arquitetura de Software**
   - Padrões de desenvolvimento
   - Capacidade de adaptação a novas frameworks
   - Fazer as perguntas certas

8. **JavaScript**
   - Única linguagem do browser
   - Ubiquidade e versatilidade
   - Ecossistema gigantesco

## Perguntas Frequentes

### 1. Porque é que o nosso curso te prepara para o mercado agora?
Foco em fundamentos + tecnologias atuais relevantes.

### 2. Porque é que o nosso curso te prepara para o mercado no futuro?
Fundações que não mudam, capacidade de adaptação.

### 3. Quais são os exemplos de outputs do curso?
Jogos, apps, web servers, bases de dados, APIs.

### 4. Quais são os estudos de mercado / estatísticas?
Mercado em crescimento, necessidade de programadores qualificados.
