# Módulo III - Web e Redes (Semanas 6-7)

## Visão Geral

**Duração:** 2 semanas  
**Foco:** Internet, Web e concorrência  
**Objetivo:** Construir apps para web e lidar com milhares de utilizadores

## Conteúdo Programático

### 1. Network Programming
- Protocolos TCP/IP
- Sockets
- HTTP/HTTPS
- APIs REST
- Comunicação cliente-servidor

### 2. World Wide Web
- História da web
- HTML/CSS/JS (introdução)
- Protocolo HTTP
- Web servers
- Navegadores

### 3. Concurrency
- Threads e processos
- Sincronização
- Deadlocks
- Race conditions
- Thread pools
- Problemas de concorrência em sistemas reais

### 4. Functional Programming
- Paradigma funcional
- Lambda expressions
- Streams avançados
- Immutability
- Funções puras

## Competências Desenvolvidas

Ao final deste módulo, os alunos serão capazes de:
- [ ] Compreender protocolos de rede
- [ ] Criar aplicações cliente-servidor
- [ ] Implementar concorrência
- [ ] Evitar problemas de sincronização
- [ ] Usar programação funcional
- [ ] Lidar com múltiplos utilizadores

## Projetos Práticos

- Chat application
- Web server básico
- Sistemas concorrentes
- API REST simples

## O Desafio da Concorrência

> "É desejável para qualquer indústria que o software tenha milhões de utilizadores, muitas vezes em simultâneo. Este é talvez ainda hoje o problema central da construção de software."

Exemplos reais:
- Sistemas de vendas de bilhetes
- Transações bancárias
- Gestão de stocks online
- Bolsa de valores e trading
