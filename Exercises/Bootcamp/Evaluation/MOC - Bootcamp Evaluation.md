# MOC: Bootcamp Evaluation Framework

## Sobre Este Documento

Framework completo para avaliação e documentação do bootcamp, focando em feedback contínuo e melhoria iterativa.

## Filosofia Central

> "It's education, not training."

Nosso objetivo não é ensinar receitas, mas fornecer ferramentas para que os alunos aprendam a aprender.

## Paradigmas Educacionais

### [[Constructivism]]
Abordagem do modelo mental — conhecimento é construído, não recebido.

### [[Behaviorism]]
Identificar o que os alunos devem aprender e dar-lhes autonomia.

### [[Cognitivism]]
Foco na teoria da carga cognitiva.

## Conceitos-Chave

### OBE - Outcome-Based Learning
Preocupação com o que os alunos conseguirão fazer depois de ensinados.

### ILO - Intended Learning Outcomes
Conjunto de skills, conhecimentos e capacidades que os alunos terão.

## Sistema de Avaliação

### [[Assessment]] (THEM)
Como estão os alunos?

### [[Evaluation]] (US)
Como estamos a ensinar?

### [[Assurance of Learning]]
O que aprenderam que não sabiam antes?
