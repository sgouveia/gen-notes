# Slides - Presentation Magic

## Introdução

"Quanto tempo gastam a formatar slides e decidir que gráficos usar?"

O Gemini vai buscar conteúdo e dados e cria apresentações profissionais automaticamente — desde estrutura até design.

---

## Demo 1: Auto-Create from Content

### Cenário
Transformar documento de preparação TechFlow em apresentação.

### Passos

**1. Novo Slides com template**
Abrir Google Slides com template da empresa já aplicado.

**2. Geração da apresentação:**

**Prompt:**
```
Cria uma apresentação de 6 slides para a reunião com a TechFlow baseada no documento de preparação que criei. Inclui: abertura, agenda, proposta de formação, benefícios, investimento, próximos passos
```

**3. Refinamento visual:**

**Prompt:**
```
No slide dos benefícios, adiciona ícones e torna mais visual
```

**4. Destaque de ROI:**

**Prompt:**
```
Muda o layout do slide do investimento para destacar o ROI
```

## Demo 2: Data Integration

### Continuação da mesma apresentação

**Integração com Sheets:**

**Prompt:**
```
No slide do investimento, inclui o gráfico de ROI que criei no Google Sheets para mostrar o retorno a 6 e 12 meses
```

**Se funcionar:**
> "Esta é a verdadeira magia — dados do Sheets integrados automaticamente!"

**Se não funcionar:**
Mostrar inserção manual com assistência Gemini (copiar valores, criar gráfico no Slides).

## Prompts Úteis para Slides

### Criar Apresentação
```
Cria uma apresentação de [N] slides sobre [tema], incluindo: [lista de secções]. Mantém o design do template atual
```

### A partir de Documento
```
Usando este template, cria uma apresentação baseada no [documento/conteúdo]. Mantém o design e cores do template
```

### Refinamento Visual
```
No slide de [tópico], adiciona [ícones/gráficos/imagens] e torna mais [visual/profissional/impactante]
```

### Integração de Dados
```
No slide de [tópico], adiciona os valores principais da tabela de [origem] que criei no Sheets
```

### Design e Layout
```
Analisa esta apresentação e sugere melhorias no design para [audiência específica]
```

### Consistência
```
Verifica se todos os slides seguem o mesmo padrão visual e [lista de elementos: cores, fontes, alinhamento]
```

## Dicas de Design com Gemini

**Estrutura sugerida:**
- Slide 1: Título e subtítulo impactante
- Slide 2: Agenda/roadmap
- Slide 3-4: Conteúdo principal (dividido visualmente)
- Slide 5: Dados/ROI/evidências
- Slide 6: Próximos passos/Call to action

**Elementos visuais:**
- Pedir ícones relevantes
- Gráficos integrados
- Layouts alternativos
- Destaques e callouts
