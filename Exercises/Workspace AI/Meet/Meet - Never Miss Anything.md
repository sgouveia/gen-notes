# Meet - Never Miss Anything

## Introdução

"Quantas reuniões participam por semana? Quantas informações importantes se perdem?"

O Gemini no Google Meet captura, resume e transforma reuniões em action items — nunca mais perca nada.

---

## Demo 1: AI-Generated Meeting Summaries

### Cenário
Reunião de vendas com TechFlow (continuação do workflow).

### Funcionalidades

**1. Transcrição ao vivo:**
- Gemini transcreve automaticamente
- Identifica diferentes oradores
- Marca timestamps importantes

**2. Resumo automático:**

**Prompt (após reunião):**
```
Resume os pontos principais discutidos nesta reunião de 45 minutos com a TechFlow
```

**3. Decisões e acordos:**

**Prompt:**
```
Identifica todas as decisões tomadas e acordos alcançados nesta reunião
```

## Prompts Úteis para Meet

### Durante a Reunião

**Resumo em tempo real:**
```
Resume o que foi discutido nos últimos [X] minutos
```

**Esclarecimento:**
```
O que foi decidido sobre [tópico específico]?
```

**Pontos pendentes:**
```
Que pontos da agenda ainda não foram cobertos?
```

### Após a Reunião

**Resumo completo:**
```
Resume os pontos principais discutidos nesta reunião de [duração] com [participantes]
```

**Decisões:**
```
Identifica todas as decisões tomadas e acordos alcançados
```

**Action items:**
```
Extrai todos os action items com: responsáveis, deadlines, prioridades
```

**Follow-up:**
```
Cria um email de follow-up para [destinatário] resumindo: discussões, decisões, próximos passos
```

### Análise Avançada

**Sentimento:**
```
Analisa o sentimento e tom desta reunião. Identifica preocupações, objeções, entusiasmo
```

**Próxima reunião:**
```
Com base nesta reunião, cria uma agenda para o próximo encontro
```

**Documentação:**
```
Adiciona este resumo à documentação do projeto [nome] no Spaces/Docs
```

---

## Dicas de Produtividade

**Antes da reunião:**
- Verificar se gravação/transcrição estão ativas
- Partilhar agenda no chat
- Preparar documentos de referência

**Durante a reunião:**
- Usar Q&A para não interromper fluidez
- Marcar pontos importantes
- Verificar action items em tempo real

**Após a reunião:**
- Revisar resumo automático
- Converter action items em tarefas
- Enviar follow-up rapidamente
- Atualizar documentação do projeto
