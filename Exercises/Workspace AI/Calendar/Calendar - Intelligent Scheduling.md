# Calendar - Intelligent Scheduling

## Introdução

"Quantas mensagens trocam só para agendar uma reunião? Quantas vezes esquecem de preparar agenda?"

O Gemini transforma o Calendar num assistente pessoal que agenda, prepara e optimiza reuniões — com integração completa.

---

## Demo 1: Smart Scheduling

### Cenário
Agendar reunião com Maria Santos (TechFlow).

### Passos

**1. Criar evento:**
Título: "Reunião TechFlow - Proposta Formação IA"

**2. Sugestão de horários:**

**Prompt:**
```
Preciso de agendar uma reunião de 1h30 com maria.santos@techflow.pt para a próxima semana, preferencialmente de manhã. Sugere 3 opções de horário e verifica a disponibilidade
```

**3. Buffers de preparação:**

**Prompt:**
```
Adiciona 30 minutos de preparação antes da reunião e 15 minutos de follow-up depois
```

## Prompts Úteis para Calendar

### Agendamento
```
Preciso de agendar uma reunião de [duração] com [email/nome] para [período], preferencialmente [preferência horário]. Sugere [N] opções de horário
```

### Buffers e Preparação
```
Adiciona [X] minutos de preparação antes da reunião e [Y] minutos de follow-up depois
```

### Geração de Agenda
```
Baseado em [contexto/documento], cria uma agenda detalhada para esta reunião de [duração]. Inclui: [tópicos específicos]
```

### Timing Estruturado
```
Divide esta agenda em blocos de tempo específicos e indica quanto tempo dedicar a cada ponto
```

### Preparação
```
Adiciona uma checklist de preparação com: documentos a levar, perguntas-chave a fazer, informações a confirmar
```

### Integração com Spaces
```
Conecta esta reunião ao projeto [nome] no Spaces e cria tarefas para [ações específicas]
```

### Lembretes
```
Programa lembretes automáticos para [ação] [tempo] antes/depois da reunião
```

### Série de Reuniões
```
Transforma isto numa série de [frequência] durante [período] para [objetivo], com agendas evolutivas
```

## Dicas de Produtividade

**Antes da reunião:**
- Buffer de preparação automático
- Documentos na descrição do evento
- Lembretes estratégicos

**Durante a reunião:**
- Agenda na descrição para fácil acesso
- Links para docs/presentações relevantes

**Após a reunião:**
- Tarefas automáticas no Spaces
- Follow-up calendarizado
- Próximos passos documentados
