# Docs - Content Creation

## Introdução

"Quantos já ficaram bloqueados com uma página em branco?"

O Google Docs com Gemini transforma o bloqueio criativo em produção fluida — de zero ao documento final em minutos.

---

## Demo 1: From Zero to Hero

### Cenário
Relatório sobre tendências de trabalho remoto para administração.

### Passos

**1. Documento em branco**
Abrir novo documento Google Docs.

**2. Geração inicial:**

**Prompt:**
```
Escreve um relatório executivo sobre as principais tendências do trabalho remoto em 2025, incluindo estatísticas, benefícios, desafios e recomendações para empresas portuguesas
```

**3. Refinamento de conteúdo:**

**Prompt:**
```
Adiciona uma secção específica sobre ferramentas digitais essenciais e respectivos custos
```

**4. Ajuste de estilo:**

**Prompt:**
```
Torna este documento mais formal e adequado para apresentar ao conselho de administração
```

**Narração:** "Vejam como estrutura automaticamente com títulos e subtítulos, informação de qualidade."

## Demo 2: Document Analysis & Summarization

### Setup
Documento técnico/regulamento de 20+ páginas preparado antecipadamente.

### Passos

**1. Upload/abrir documento longo**
Carregar PDF ou documento extenso.

**2. Resumo executivo:**

**Prompt:**
```
Resume este documento em 5 pontos principais e identifica 3 acções que a nossa empresa deve tomar
```

**3. Checklist de compliance:**

**Prompt:**
```
Cria uma checklist das obrigações legais mencionadas neste documento
```

**4. Análise avançada:**

**Prompt:**
```
Identifica potenciais riscos ou oportunidades que este documento apresenta para PMEs portuguesas
```

## Demo 3: Cumulative Integration

### Cenário
Preparação para reunião TechFlow (conectando com email da abertura).

### Passos

**1. Novo documento**
Criar documento de preparação.

**2. Agenda baseada em email:**

**Prompt:**
```
Com base no email da Maria Santos da TechFlow que recebi hoje, cria uma agenda detalhada para a nossa reunião. Inclui: objetivos da reunião, perguntas a fazer sobre a equipa deles, pontos a clarificar sobre orçamento e cronograma
```

**Fallback** (se integração não funcionar):
```
A TechFlow tem 50 pessoas na equipa de marketing, querem formação em IA para criação de conteúdo, automação e análise de dados. Cria uma agenda para reunião de vendas
```

**3. Perguntas técnicas:**

**Prompt:**
```
Adiciona uma secção com perguntas técnicas específicas sobre o nível atual de conhecimento da equipa deles
```

## Prompts Úteis para Docs

### Criar do Zero
```
Cria um [tipo de documento] sobre [tema], incluindo: [secções específicas]. Tom: [formal/informal/consultivo]
```

### Expandir Conteúdo
```
Expande a secção sobre [tópico] com [exercícios práticos/exemplos específicos/casos de estudo]
```

### Analisar Documento
```
Analisa este documento e identifica: pontos-chave, acções recomendadas, riscos, oportunidades
```

### Adaptar para Audiência
```
Torna este documento adequado para [audiência específica], ajustando tom, vocabulário e profundidade
```
