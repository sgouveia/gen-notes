# Sheets - Data Intelligence

## Introdução

"Quem já perdeu tempo a fazer fórmulas complexas ou a tentar perceber que gráfico fazer?"

O Gemini transforma o Sheets num analista de dados pessoal que entende português simples.

---

## Demo 1: Natural Language Queries

### Setup
Sheet com dados de vendas mensais, por região (Norte/Centro/Sul), por produto (A/B/C).

### Queries Progressivas

**1. Query simples:**
```
Qual foi o mês com melhores vendas?
```

**2. Query média:**
```
Compara as vendas entre Norte e Sul no último trimestre
```

**3. Query avançada:**
```
Identifica tendências sazonais e prevê vendas para os próximos 3 meses
```

**Narração:** "Isto são horas de análise transformadas em segundos. Estou a falar em português normal, não em Excel!"

## Demo 3: Cumulative Integration - TechFlow Analysis

### Cenário
Análise de ROI para proposta TechFlow (conectando com Docs e contexto do webinar).

### Passos

**1. Nova sheet/aba**
Criar nova aba na sheet existente ou nova spreadsheet.

**2. Análise de ROI:**

**Prompt:**
```
Cria uma tabela com análise de ROI para formação de 50 pessoas em IA. Inclui: investimento inicial, horas poupadas por pessoa/mês, valor/hora médio em Portugal, ROI a 6 e 12 meses
```

**3. Visualização para apresentação:**

**Prompt:**
```
Transforma isto num gráfico que posso usar na apresentação à TechFlow
```

## Prompts Úteis para Sheets

### Análise Básica
```
Qual foi o [máximo/mínimo/média] de [métrica] em [período]?
```

### Comparações
```
Compara [métrica A] com [métrica B] no período de [datas]
```

### Tendências e Previsões
```
Identifica tendências em [dados] e prevê [resultado] para [período futuro]
```

### Visualizações
```
Cria um gráfico de [tipo] mostrando [dados] com [formatação específica]
```

### Dashboards
```
Cria um dashboard com os [N] KPIs mais importantes para [objetivo de negócio]
```

### Análise de ROI
```
Calcula ROI para [projeto/investimento] incluindo: custos, benefícios, payback, [período]
```
