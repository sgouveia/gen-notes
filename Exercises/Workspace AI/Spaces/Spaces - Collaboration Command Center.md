# Spaces - Collaboration Command Center

## Introdução

"Quantos projetos têm espalhados entre emails, mensagens, documentos e reuniões? Como fazem follow-up de tarefas e decisões?"

O Spaces com Gemini é o centro de comando inteligente que organiza tudo automaticamente e nunca deixa nada passar.

---

## Demo 1: Smart Project Setup

### Cenário
Criar espaço colaborativo projeto TechFlow.

### Passos

**1. Novo Space:**
Nome: "Projeto TechFlow - Formação IA Marketing"

**2. Estrutura organizada:**

**Prompt:**
```
Organiza este espaço para gerir um projeto de formação em IA para uma empresa de 50 pessoas. Cria secções para: documentação, cronograma, comunicação com cliente, materiais de formação, follow-up e avaliação
```

**3. Seed de conteúdo:**

**Prompt:**
```
Adiciona ao espaço: a proposta que criámos no Docs, o orçamento do Sheets, a apresentação do Slides, e o evento agendado no Calendar
```

**4. Descrição e objetivos:**

**Prompt:**
```
Cria uma descrição inteligente do projeto e define objetivos SMART automáticos baseados no conteúdo que já temos
```

## Demo 2: Intelligent Task Management

### Continuação no mesmo Space

**1. Criação de tarefas:**

**Prompt:**

**2. Priorização AI:**

**Prompt:**
```
Analisa estas tarefas e sugere a ordem de prioridade baseada em impacto no projeto e urgência
```

**3. Identificação de bottlenecks:**

**Prompt:**
```

**Prompt:**
```

**Prompt:**
```
Cria métricas automáticas para acompanhar progresso: % conclusão, tarefas em risco, milestone tracking
```

## Prompts Úteis para Spaces

### Setup de Projeto
```
Organiza este espaço para gerir [tipo de projeto]. Cria secções para: [lista de categorias/áreas]
```

### Importação de Conteúdo
```
Adiciona ao espaço: [lista de recursos: docs, sheets, slides, eventos, emails]
```

### Definição de Objetivos
```
Cria uma descrição do projeto e define objetivos [SMART/específicos] baseados em [contexto/documentação]
```

### Gestão de Tarefas
```
Baseado em [cronogramo/plano], cria uma lista completa de tarefas com: responsáveis, deadlines, dependências
```

### Priorização
```
Analisa estas tarefas e sugere ordem de prioridade baseada em [critérios: impacto, urgência, recursos]
```

### Identificação de Problemas
```
Identifica potenciais [bottlenecks/risks/conflitos] e sugere [tarefas paralelas/mitigações]
```

### Sincronização Calendar
```
Conecta as tarefas [específicas/todas] com o calendário para alocação de tempo
```

### Tracking de Progresso
```
Cria métricas para acompanhar: % conclusão, tarefas em risco, milestones, [outros KPIs]
```

### Resumo de Discussões
```
Resume as últimas [N] mensagens e identifica: decisões, questões pendentes, próximos passos
```

### Action Items
```
Extrai action items desta discussão e converte em tarefas atribuídas com [deadlines/prioridades]
```

### Relatórios Automáticos
```
Cria um resumo [frequência: semanal/mensal] do progresso para [audiência: equipa/stakeholders]
```

## Dicas de Organização

**Estrutura recomendada:**
- Tasks/Tasks por pessoa
- Docs/Documentação
- Discussions/Decisões
- Milestones/Entregas
- Resources/Recursos

**Integrações úteis:**
- Calendar para deadlines
- Docs para documentação
- Sheets para tracking
- Meet para reuniões
