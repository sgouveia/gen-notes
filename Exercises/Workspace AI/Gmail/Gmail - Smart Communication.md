# Gmail - Smart Communication Assistant

## Introdução

O Gemini no Gmail não é só autocompletar — é ter um assistente de comunicação profissional que entende contexto, tom e audiência.

**Pergunta-chave:** "Quantos emails mandam por dia? Quantos demoram mais de 5 minutos a escrever?"

---

## Demo 1: Smart Compose vs Help Me Write

### Cenário
Responder a cliente insatisfeito com atraso na entrega.

**Email problema:**
> "Caro João, o projeto devia ter sido entregue ontem e ainda não recebemos nada. Isto está a afetar o nosso cronograma. Esperamos uma resposta urgente. Cumprimentos, Pedro Silva"

### Passos

1. **Smart Compose básico** - Sugestões automáticas enquanto escreve

2. **Help Me Write avançado:**

**Prompt:**
```
Responde de forma profissional mas empática, reconhecendo o atraso, explicando que houve um imprevisto técnico, apresentando uma nova data de entrega concreta, e oferecendo uma compensação
```

**Refinamentos:**
- "Torna isto mais pessoal e menos corporativo"
- "Adiciona mais senso de urgência e responsabilidade"

## Demo 2: Diferentes Tons e Audiências

### Cenário
Comunicar aumento de preços 15% — mesmo conteúdo, 3 audiências diferentes.

### Versão 1: Cliente Corporativo

**Prompt:**
```
Comunica um aumento de 15% nos preços devido à inflação, para um cliente empresarial, de forma profissional e justificada
```

### Versão 2: Pequeno Negócio

**Prompt:**
```
Mesmo conteúdo mas para um pequeno empresário, tom mais próximo e empático
```

### Versão 3: Utilizador Individual

**Prompt:**
```
Mesmo conteúdo mas para cliente individual, mais informal e personalizado
```

**Nota:** Mostrar lado a lado as três versões para comparar adaptação de tom.

## Demo 3: Email Summarization

### Cenário
Thread longa com múltiplas pessoas sobre organização de evento (15+ emails).

### Passos

1. **Resumir conversa:**

**Prompt:**
```
Resume os pontos principais desta conversa e identifica que decisões ainda precisam ser tomadas
```

2. **Criar action items:**

**Prompt:**
```
Cria uma lista de ação com responsáveis baseada nesta thread
```

## Hands-on com Voluntários

**Desafio:** Pedir à audiência uma situação real de comunicação difícil.

**Execução:**
1. Coletar problema da audiência
2. Executar prompt sugerido em tempo real
3. Mostrar refinements e ajustes de tom
4. Demonstrar iteração rápida

---

## Prompts Úteis para Gmail

### Resposta Profissional
```
Ajuda-me a redigir uma resposta profissional a este email, agradecendo o interesse e pedindo [X] questões de esclarecimento sobre: [tópicos]
```

### Refinamento de Tom
```
Torna esta resposta mais [adjétivo: consultiva/direta/empática/formal] e adiciona uma frase que demonstre [experiência/confiança/compreensão]
```

### Resumo de Thread
```
Resume os pontos principais desta conversa de [N] emails e identifica: decisões tomadas, decisões pendentes, e próximos passos
```
