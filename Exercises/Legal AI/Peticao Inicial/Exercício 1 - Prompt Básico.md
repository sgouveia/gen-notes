# Exercício: Petição Inicial - Prompt Básico

## Objetivo
Explorar o que acontece quando usamos o prompt mais simples possível para gerar uma petição inicial.

## O Prompt

```
O prompt mais básico possível irá ser o que apenas anexa o enunciado e pede uma petição inicial.
```

**Instruções:**
1. Abra o ChatGPT, Claude, ou outro LLM
2. Cole o enunciado do caso (veja [[Enunciado - Violação de Patente]])
3. Adicione apenas: "Elabora uma petição inicial a partir deste enunciado"
4. Observe o resultado

## O que observar

### Pontos a analisar:
- [ ] A petição inclui todos os elementos necessários?
- [ ] A estrutura está correta (Factos → Direito → Pedido)?
- [ ] Há placeholders ou campos em falta?
- [ ] A fundamentação jurídica é adequada?
- [ ] Os valores estão corretamente quantificados?

### Reflexão
Este exercício demonstra os limites de prompts básicos. Use-o como baseline para comparar com as abordagens mais sofisticadas nos próximos exercícios.

## Exemplo de Output Esperado

Ver arquivo original: `AI-Legal-Exercises-Peticao-Inicial.md` (secção 1.a)

**Principais problemas identificados:**
- Pode omitir elementos estruturais importantes
- Valores e datas aparecem como placeholders

- Falta especificidade técnica sobre a violação
