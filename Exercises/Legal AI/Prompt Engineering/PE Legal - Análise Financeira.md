# PE Legal - Análise Financeira e Documentos

## Análise de Documentos Financeiros

### Cenário
Análise de relatório financeiro da IP (Infraestruturas de Portugal).

---

### 1. Análise Financeira

**Prompts:**
```
Quais foram as principais fontes de receita da IP no 1.º semestre de 2024?
```

```
Como variaram os custos operacionais face ao mesmo período de 2023?
```

```
Qual o resultado líquido e que fatores mais contribuíram para esse resultado?
```

```
A IP apresentou lucro ou prejuízo neste semestre?
```

```
Qual o nível de endividamento da empresa e houve alteração significativa?
```

---

### 2. Investimentos e Projetos

**Prompts:**
```
Que projetos ferroviários receberam mais investimento no período?
```

```
Qual o total investido em manutenção da rede rodoviária nacional?
```

```
Existem projetos em atraso? Quais as causas apresentadas?
```

```
Houve lançamento de novas obras ou concursos públicos relevantes?
```

---

### 3. Sustentabilidade e Responsabilidade Social

**Prompts:**
```
Que medidas ambientais foram implementadas pela IP no 1.º semestre de 2024?
```

```
A empresa está a cumprir as metas de sustentabilidade definidas para o ano?
```

```
Quais os indicadores de impacto ambiental reportados no documento?
```

---

### 4. Interpretação e Síntese

**Prompts:**
```
Resume os principais resultados financeiros da IP em três parágrafos.
```

```
Quais são os três principais riscos estratégicos identificados pela administração?
```

```
O relatório sugere mudanças na estratégia da empresa? Em que sentido?
```

---

### 5. Comparação com Outros Anos

**Prompts:**
```
Como se comparam os investimentos em ferrovia em 2024 com os de 2023?
```

```
Houve melhoria ou agravamento da performance operacional?
```

```
Quais indicadores mostram evolução positiva face ao 1.º semestre de 2023?
```

---

### 6. Perspectiva Crítica / Simulada com IA

**Prompts:**
```
Que recomendações darias à administração com base nos dados apresentados?
```

```
Se fosses preparar uma auditoria interna, que áreas considerarias prioritárias?
```

---

## Exercícios Práticos

### Exercício 1
Analisar relatório anual de empresa cotada.

### Exercício 2
Comparar relatórios de dois anos consecutivos.

### Exercício 3
Identificar riscos e oportunidades para investidores.
