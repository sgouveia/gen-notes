# PE Legal - Análise de Acórdãos

## Técnicas de Exploração

### 1. Resumo e Classificação

**Prompts:**
```
"Resuma este acórdão em 3 pontos principais."
```

```
"Classifique este acórdão numa das seguintes áreas do direito: 
penal, fiscal, do trabalho, sucessório, das sociedades, da família."
```

---

### 2. Extração de Informação

**Prompts:**
```
"Identifique os princípios jurídicos aplicados neste acórdão."
```

```
"Quais são as consequências práticas desta decisão para processos futuros 
de partilha de bens com imóveis no estrangeiro?"
```

---

### 3. Simplificação e Explicação

**Prompts:**
```
"Explique o acórdão em linguagem simples para um cliente que não tem 
conhecimentos jurídicos."
```

```
"O que significa o princípio da unidade e universalidade da partilha neste contexto?"
```

---

## Criação de Estratégias

### Análise de Estratégia Jurídica

**Prompts:**
```
"Se fosse o advogado da parte contrária, como argumentaria para excluir 
os bens situados no estrangeiro do inventário?"
```

```
"Como poderia este acórdão ser usado como precedente em casos semelhantes?"
```

---

### Planeamento Prático

**Prompt:**
```
"Elabore uma recomendação para um cliente com bens no estrangeiro sobre 
como proceder em processos de divórcio."
```

---

## Interpretação e Criação de Documentos

### Análise Comparativa

**Prompts:**
```
"Compare os argumentos do tribunal de primeira instância com os do 
Supremo Tribunal de Justiça."
```

```
"Quais são as diferenças entre este acórdão e a jurisprudência internacional 
sobre partilhas de bens no estrangeiro?"
```

---

## Exemplo Completo

### Cenário
Acórdão sobre partilha de bens em divórcio internacional.

### Prompt Completo
```
Analisa este acórdão sobre partilha de bens em contexto de divórcio internacional:

1. Resuma em 3 pontos principais
2. Identifique os princípios jurídicos aplicados
3. Explique as implicações para casos futuros semelhantes
4. Compare com jurisprudência anterior
5. Sugira estratégia para advogado da parte contrária
6. Elabore recomendação prática para o cliente

Formata a resposta como parecer jurídico estruturado.
```
