# Anatomia de Prompts Jurídicos

## Estrutura Eficaz

### Elementos Essenciais

1. **Contexto Jurídico**
   - Jurisdição (Portugal)
   - Área do direito
   - Legislação aplicável

2. **Persona Profissional**
   - "És um advogado especializado em..."
   - Nível de experiência
   - Contexto profissional

3. **Instrução Clara**
   - O que fazer
   - Formato esperado
   - Elementos obrigatórios

4. **Restrições Específicas**
   - Referências legais necessárias
   - Tom/formalidade
   - Extensão

---

## Exemplo Prático

### Cenário
Um inquilino não paga renda há três meses.

### Prompt Estruturado
```
És um advogado especializado em direito imobiliário. 
Resume os recursos legais disponíveis para o senhorio. 
Fornece a opinião jurídica estruturada com referências legais relevantes.
```

### Análise
- **Persona:** Advogado especializado em direito imobiliário
- **Tarefa:** Resumir recursos legais
- **Contexto:** Inquilino em incumprimento
- **Formato:** Opinião jurídica estruturada
- **Requisito:** Referências legais

## Técnicas por Categoria

### Explicações
Adaptar complexidade ao público-alvo.

**Exemplo:**
```
Explica-me a teoria da relatividade como se eu fosse um advogado 
especializado em Direito Civil.
```

**Exemplo:**
```
Responsabilidade civil extracontratual (explicado a um adulto sem formação jurídica)
```

**Exemplo:**
```
O princípio do contraditório (explicado a um estudante de engenharia)
```

### Análise de Texto Jurídico

**Fonte:** Artigo académico
https://journals.openedition.org/rccs/619

**Prompt:**
```
Analisa este artigo e extrai:
- Princípios jurídicos discutidos
- Implicações práticas
- Críticas aos argumentos apresentados
- Aplicabilidade ao ordenamento jurídico português
```

---

### Classificação de Casos

**Fonte:** Notícia jurídica
https://www.jn.pt/...

**Prompt:**
```
Analisa esta notícia jurídica e:
1. Identifica a área do direito envolvida
2. Extrai os factos relevantes
3. Identifica questões jurídicas em debate
4. Sugere fundamentação legal aplicável
```

## Checklist de Qualidade

Antes de usar um prompt jurídico:

- [ ] Jurisdição especificada?
- [ ] Área do direito clara?
- [ ] Formato definido?
- [ ] Referências legais solicitadas?
- [ ] Tom apropriado (formal/técnico)?
- [ ] Contexto suficiente?
