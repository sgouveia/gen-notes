## Templates e Frameworks

### Estrutura de Prompt Legal
```
[ROLE] → Definir expertise jurídica
[CONTEXT] → Especificizar jurisdição e área
[TASK] → Descrever tarefa concreta
[CONSTRAINTS] → Requisitos obrigatórios
[OUTPUT FORMAT] → Formato desejado
[QUALITY CRITERIA] → Critérios de sucesso
```

### Checklist de Review
- [ ] Dados completos (sem placeholders)
- [ ] Fundamentação jurídica adequada
- [ ] Estrutura processual correta
- [ ] Pedidos concretos e quantificados
- [ ] Linguagem formal e técnica
- [ ] Pronto para apresentação

## Como Usar Este Exercise Bank

### Para Auto-Estudo
1. Escolha um tópico de interesse
2. Siga a progressão de exercícios (1→2→3→4)
3. Pratique com casos reais (adaptando os enunciados)
4. Documente os seus próprios prompts eficazes

### Para Ensino
1. Use como base para workshops (2-4 horas por tópico)
2. Divida alunos em grupos para comparar abordagens
3. Peça análise crítica entre pares
4. Construa biblioteca de prompts da turma

### Para Consultoria/Prática
1. Adapte os exercícios aos seus casos reais
2. Crie templates reutilizáveis
3. Mantenha biblioteca de prompts validados
4. Itere com base em feedback de tribunais
