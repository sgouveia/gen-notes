# Prompts - Explicações

## Categoria: Explicar Conceitos

### Prompt 1: Direito do Trabalho

**Contexto:**
Muitos seguidores estão a deixar de interagir com os conteúdos nas redes sociais da marca. És o responsável de comunicação digital.

**Prompt:**
```
Cria um plano para aumentar a interação nas redes sociais (comentários, partilhas, gostos) 
de forma orgânica. Escreve um discurso motivacional que poderias dar à equipa de 
comunicação na próxima reunião.
```

---

### Prompt 2: Explicar a Outros Profissionais

**Exercício:** Explicar conceitos complexos a audiências específicas.

**Exemplo 1:**
```
Qual é o máximo possível de renovações num contrato de trabalho a termo certo?
```

**Exemplo 2:**
```
Explica-me a teoria da relatividade como se eu fosse um jornalista desportivo
```

**Exemplo 3:**
```
Explica o que é a Web 3.0 a um advogado habituado a lidar com contratos e propriedade.
```

**Exemplo 4:**
```
O que é o machine learning? Explica-me com uma analogia ligada ao Marketing.
```

---

### Prompt 3: Adaptar ao Público

**Exercício:** Adaptar complexidade ao nível do público.

**Contexto:** Campanha Viral

**Prompt:**
```
Campanha Viral (explicado a um adulto sem formação em marketing)
```

**Outro exemplo:**
```
O princípio do contraditório (explicado a um estudante de engenharia)
```

---

## Técnica: Persona + Contexto

### Estrutura
1. **Definir a persona** (quem explica)
2. **Definir o público** (para quem explica)
3. **Escolher o conceito** (o quê)
4. **Especificar o formato** (como)

### Exemplo Completo
```
Persona: Expert em direito do trabalho
Público: Estudante de engenharia (sem formação jurídica)
Conceito: Princípio do contraditório
Formato: Explicação simples com analogia
```

**Output esperado:**
Explicação acessível usando conceitos familiares ao estudante de engenharia.
