# Prompts - Análise de Texto e Classificação

## Análise de Texto

### Exemplo Prático

**Fonte:**
https://journals.openedition.org/rccs/619

**Prompt:**
```
Analisa o artigo no link fornecido e extrai:
- Ideias principais
- Argumentos centrais
- Conclusões
- Implicações para comunicação estratégica
```

**Aplicação:**
- Análise de literatura académica
- Síntese de artigos longos
- Pesquisa para relatórios

---

## Classificação

### Exemplo Prático

**Fonte:**
Notícia do JN: https://www.jn.pt/6381118287/...

**Prompt:**
```
Analisa esta notícia e:
1. Resume em 3 pontos principais
2. Identifica o ângulo jornalístico
3. Classifica o tipo de conteúdo (facto, opinião, investigação)
4. Sugere ângulos alternativos para a história
```

**Aplicação:**
- Monitorização de media
- Análise de concorrência
- Identificação de oportunidades de comunicação

---

## Técnicas de Análise

### 1. Extração de Informação

**Prompt:**
```
Extrai do texto:
- Entidades mencionadas (pessoas, organizações)
- Dados numéricos relevantes
- Datas importantes
- Citações diretas
- Sentimento geral
```

### 2. Sumarização

**Prompt:**
```
Resume este texto em:
- 1 parágrafo (100 palavras)
- 3 bullets principais
- 1 tweet (280 caracteres)
```

### 3. Análise de Sentimento

**Prompt:**
```
Analisa o sentimento deste texto:
- Positivo/Negativo/Neutro
- Tons específicos (entusiasmo, preocupação, etc.)
- Público-alvo implícito
```

## Exercícios Práticos

### Exercício 1: Análise de Comunicado
Enviar um comunicado de imprensa e pedir:
- Análise de estrutura
- Identificação de key messages
- Sugestões de melhoria

### Exercício 2: Benchmarking
Analisar 3 comunicados da concorrência e comparar:
- Tom de voz
- Estrutura
- Eficácia

### Exercício 3: Crisis Analysis
Analisar notícias negativas e identificar:
- Pontos de controvérsia
- Resposta recomendada
- Stakeholders a considerar
