# Introdução à IA Generativa

## LLMs e Ferramentas

### O Que São LLMs?

**Large Language Models** - Modelos de linguagem de grande escala treinados com vastas quantidades de texto.

**Capacidades:**
- Compreensão de linguagem natural
- Geração de texto
- Resposta a perguntas
- Análise de sentimento
- Tradução
- Sumarização

### Principais Ferramentas

#### 1. Claude (Anthropic)
- Foco em segurança e alinhamento
- Excelente para análise longa
- Contexto extenso

#### 2. GitHub Copilot
- Integração com IDEs
- Foco em código
- Autocompleção inteligente

#### 3. Gemini (Google)
- Integração com Google Workspace
- Multimodal (texto, imagem, áudio)
- Acesso a informação atualizada

#### 4. ChatGPT (OpenAI)
- Versão mais popular
- GPT-4 para tarefas complexas
- Plugins e extensões

#### 5. LLaMA (Meta)
- Open source
- Customizável
- Custo reduzido

#### 6. Amazon Bedrock
- Infraestrutura AWS
- Múltiplos modelos
- Escalabilidade empresarial

### Demonstração Prática

**Prompt de Abertura:**
```
Conta-me uma história engraçada e bem intencionada sobre um jovem 
comunicador que começou a trabalhar numa grande agência de comunicação.
```

**Objetivo:** Mostrar capacidade criativa e tom apropriado.
